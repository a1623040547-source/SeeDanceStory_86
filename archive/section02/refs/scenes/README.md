# 场景图

本文件夹用于存放**项目启动与生成时所需的场景/首图**。

## 放置内容

- **段 1 用首图**：辛恩与蕾娜在客厅沙发、茶几、书架、窗外雨天、暖光，86 动画风。可作为图生视频的唯一输入图（即 `section02_ref_img.jpg` 的源文件或副本）。
- 可选：纯场景图（无人物或仅作布局参考），用于与 `refs/characters/` 中的角色图合成首图。

## 使用方式

- 段 1 生成时，将本文件夹内的首图（含两人+场景）上传至 Seedance **图生视频**的图片输入。
- 生成完成后，段 2～12 使用的尾帧图（tail_01～tail_11）由各段生成视频截取，不放在本文件夹。

## 命名建议

- `section02_ref_img.jpg`（段 1 用首图）
- 或 `section02_scene_living_room.png`
