# 环境氛围参考图

本文件夹用于存放**第一段（段 1）确立的环境/氛围参考图**，供段 2～12 生成时锁定与段 1 一致的打光、窗外天气与整体色调，避免随 phase 漂移成下雪、冷光或画面变花。

## 背景（section02 后期复盘）

- 仅靠「上一段尾帧 → 下一段」V2V 衔接，能保证人物与动作连续，但**整体环境、打光、窗外天气**会随 phase 往后漂移。
- 表现：前段暖光、窗外下雨，后段易变为下雪或冷光，画面变花。因此需要**统筹全片环境/氛围**，并通过额外 ref 固定。

## 放置内容

- **一张环境氛围参考图**：来自段 1 的典型帧或首图，能代表本 section 的**暖光、雨窗、室内氛围**（86 动画风）。
- 建议从段 1 生成视频中截取一帧（如中段或尾帧中光线与窗外雨清晰的画面），或直接使用段 1 首图/尾帧，保存至此。

## 使用方式

- **段 1 生成后**：从 `phases/phase01/phase01.mp4` 中截取一帧，或使用 `refs/scenes/` 中的首图，保存为 `section02_env_ref.png` 放入本文件夹。
- **段 2～12 生成时**：若 Seedance 支持多图输入，除上传「上一段尾帧」外，**额外上传本文件夹中的环境氛围参考图**，作为「环境/场景/打光」参考；并在提示词中保持「same warm lighting, rainy window, no snow」等锚定句。

## 命名建议

- `section02_env_ref.png`（或 `.jpg`）

## 与 refs 其他文件夹的关系

- **refs/characters/**：角色形象，全程不变。
- **refs/scenes/**：段 1 首图（场景+人物），仅段 1 使用。
- **refs/env_atmosphere/**：段 1 确立的环境/打光/窗外天气，段 2～12 用作氛围参考，与尾帧一起保证「人物衔接 + 环境一致」。
