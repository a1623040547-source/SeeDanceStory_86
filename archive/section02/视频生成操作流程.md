> 86 辛恩×蕾娜，从首图到成片的完整步骤说明，适用于 Seedance 2.0。  
> **提示词**：每段 `phases/phaseXX.txt` 内已有**可直接整段复制到 Seedance 2.0** 的完整英文提示词（含本段画面与台词概述），无需再拼接总述。  
> **素材与衔接**：详见 `00-项目总述与素材说明.md`。

## 二、段 1：图 + 文生视频（仅此段用图生）

| 步骤 | 操作 |
|:----:|------|
| 1 | 在 Seedance 2.0 中选择 **图生视频**（或 Image + Text to Video）。 |
| 2 | **上传首图**：上传「首图」`section02_ref_img.jpg`（**不是**角色参照图；此图=场景+两人定调，仅段 1 使用）。 |
| 3 | **时长**：设为 **15 秒**。 |
| 4 | **提示词**：打开 `phases/phase01/phase01.txt`，将「以下整段复制到 Seedance 2.0 提示词框」下方的**整段英文**复制到 Seedance 提示词框即可（已含 86 风格、辛恩蕾娜、本段画面与台词）。 |
| 5 | **负面词**：在同一文件中复制「负面词」整段到 Seedance 负面词框。 |
| 6 | **比例**：原比例（或与首图一致）。 |
| 7 | 生成并下载视频，命名为如 `section02_seg01.mp4`。 |
| 8 | **截取尾帧**：必须截取该段视频的**最后一帧**，导出为 `section02_tail_01.png`。此图为**上一段尾帧图**，供段 2 作为 V2V 的视觉起点，以保证与下一段连贯；勿与首图、角色参照图混淆。 |

## 三、段 2～12：V2V 接力（用上一段尾帧）

每一段都采用「**上一段尾帧图** + 总述 + 本段提示词」做 **Video-to-Video（V2V）** 生成。尾帧图=上一段结尾画面，用于与下一段准确衔接，**不是**角色参照图。

| 步骤 | 操作 |
|:----:|------|
| 1 | 在 Seedance 2.0 中选择 **V2V**（或 Video-to-Video / 以图/视频为参考）。 |
| 2 | **上传参考**：上传 **上一段尾帧图**（段 2 用 `tail_01.png`，段 3 用 `tail_02.png`，…，段 12 用 `tail_11.png`）。若平台支持「上一段完整视频」并可指定取最后一帧，可同时上传上一段视频；否则仅用尾帧图即可。 |
| 3 | **时长**：**15 秒**。 |
| 4 | **V2V 强度**：建议 **30%～40%**，过强易崩脸、过弱动作不明显，可先试 35%。 |
| 5 | **提示词**：打开当前段对应的 `phases/phaseXX.txt`，将「以下整段复制到 Seedance 2.0 提示词框」下方的**整段英文**复制到 Seedance 提示词框即可（已含 86 风格、辛恩蕾娜、本段画面与台词）。 |
| 6 | **负面词**：在同一文件中复制「负面词」整段到 Seedance 负面词框。 |
| 7 | 生成并下载，命名如 `section02_seg02.mp4`～`section02_seg12.mp4`。 |
| 8 | **为下一段准备尾帧**：从本段视频截取**最后一帧**，保存为对应的 `section02_tail_02.png`～`section02_tail_11.png`（段 12 无需再截）。下一段生成时上传的正是这张尾帧图，以保证与下一段连贯。 |

**循环**：对段 3～12 重复上述「上传上一段尾帧图 → 总述+本段提示词 → 生成 → 截尾帧」，直到 12 段全部生成完毕。

## 四、导出与剪辑成片

| 步骤 | 操作 |
|:----:|------|
| 1 | **按时间轴拼接**：在剪辑软件（如 DaVinci Resolve / Premiere / 剪映）中，按 README「四、剪辑总时间轴」顺序导入 `section02_seg01.mp4`～`section02_seg12.mp4`，首尾相接，总长 3 分钟。 |
| 2 | **对齐配音**：若有日文配音，按每段 0～15s、15～30s、…、2:45～3:00 对齐到对应画面。 |
| 3 | **统一调色/音量**：可对 12 段做轻微色彩统一与音量平衡，使观感一致。 |
| 4 | **导出成片**：输出为最终 MP4（建议码率与分辨率与单段一致或更高）。 |

## 五、常见问题与防崩脸提醒

| 问题 | 建议 |
|------|------|
| **脸型/画风突变** | 每段提示词中务必保留「consistent character」「86 anime style」及场景词；V2V 强度不要超过 40%。 |
| **动作跳跃** | 尾帧必须是**视频最后一帧**，才能保证下一段开头与上一段结尾连贯。 |
| **时长不足 15s** | 确认 Seedance 2.0 当前版本单段上限为 15s，若平台限制更短，需在 README 中把每段时长改为实际最大值并相应增加段数。 |
| **段数/总时长变更** | 若改为 10 段或 20 段，需同步修改 README 中的「逐段剧本」与「剪辑总时间轴」，并相应调整本流程中的段号与命名。 |
