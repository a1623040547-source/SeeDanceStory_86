# section04 参考图

本目录存放段 1 及生成时使用的**场景图**与**角色图**。文件名请按下列规范，便于脚本与文档引用。

## 命名规范总览

| 目录 | 规范文件名 | 说明 |
|------|------------|------|
| **scenes/** | `section04_scene_01.png` | 海边场景（可无人物） |
| **scenes/** | `section04_ref_img.jpg` | 可选：段 1 首图（场景+两人） |
| **characters/** | `section04_character_01.png` | 角色参考 1 |
| **characters/** | `section04_character_02.png` | 角色参考 2 |
| **characters/** | `section04_character_couple.png` | 可选：两人同框 |
| **refs/**（根） | `section04_ref_composite.jpg` | 可选：多图合成 2×2（仅作参考，不能当首帧） |

**避免**：即梦等平台的长文件名（如 `jimeng-2026-02-19-xxxx-描述....png`），放入本目录后请重命名为上表格式。

详见各子目录下 README。
