# section03 中期复盘

> 复盘时间：基于当前已生成段 1～7 的成片观感。

---

## 一、问题：男主「沉稳寡言」被呈现为无反应、无互动

| 项目 | 内容 |
|------|------|
| **人设设定** | 辛恩：沉稳、寡言（少台词、不张扬） |
| **视频中的表现** | 男主几乎没有微表情，神态/情绪/肢体动作都没有和女主产生互动，观感像「木头人」 |
| **根因** | 「寡言」被误读为「无反应」；剧本与提示词中辛恩的「对蕾娜的回应」不够显式，模型易生成「少动、少表情」的静态男性 |

---

## 二、改进方向（与《section-phase-script-retro》动作/神态/互动一致）

### 1. 人设澄清（写入 00 / README 或本 section 规则）

- **沉稳寡言** = 少台词、不夸张、不抢戏，**不等于**无表情、无反应、无互动。
- 辛恩应有对蕾娜的**持续微反应**：听她说话时的视线/眼珠微动、喉结微动、呼吸/肩动、手指/栏杆上的小动作、被她逗到时眉或嘴角的极轻微变化、不躲开她的靠近等。
- 原则：**蕾娜有动作/台词 → 辛恩必须有可被画面执行的回应**（哪怕很克制）。

### 2. 秒级表「青年」列必须写「对蕾娜的回应」

- 每一段的秒级表里，「青年（辛恩）」列不能只有「保持望夜景」「同上一格」「静止」等被动描述。
- 需明确写出**触发–反应**：蕾娜做 X 的时段内，辛恩的 Y 反应（如：她说「哼—」→ 他眼珠向她动约 5 度、喉结微动；她靠过来 → 他手指在栏杆上微动、不躲）。
- 微反应类型要具体：眉/眼/嘴角/喉结、视线方向变化、手指/肩/呼吸、下颌线放松等。

### 3. 提示词【本段动作与节奏】显式写辛恩的互动节奏

- 粘贴到 Seedance 的【本段动作与节奏】不能只强调蕾娜的动作与台词，**必须交替或成对写出「少女做 A → 青年做 B」**。
- 示例（改进前 vs 改进后）：
  - 改进前：少女说哼—侧头看他，青年望夜景。少女看他侧脸笑意明显。
  - 改进后：少女说哼—侧头看他，**青年眼珠微向她动、喉结微动仍不转头**。少女看他侧脸笑意明显，**青年望夜景呼吸一次、肩微动**。
- 目的：让模型明确「男主有在听、在看、在反应」，只是用克制的微表情与肢体表达。

### 4. 适用范围与执行顺序

| 范围 | 建议 |
|------|------|
| **未生成 / 待改的 phase（8～12）** | 写或改剧本时：秒级表「青年」列补足对蕾娜的回应；【本段动作与节奏】中成对写少女→青年反应；【项目·风格·规则】或 00 中可加一句「青年沉稳寡言但有微表情与对少女的肢体/视线反应」。 |
| **已生成的 phase（1～7）** | 若重制：按上述要点加强辛恩的微表情与互动反应后再生成。 |
| **全 section 统一** | 在 00 或 README 的「角色/风格」中补一句：辛恩沉稳寡言但需有对蕾娜的微表情与肢体/视线互动，避免成片呈现为无反应。 |

## 四、与《section-phase-script-retro》的对应

- 本复盘对应 Skill 中 **「四、动作/神态/细节描写」**：两人互动时写清**触发与反应**（A 做 X → B 做 Y）。
