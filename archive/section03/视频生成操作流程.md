# 《86｜不存在的日常》露台·傍晚小剧场 视频生成操作流程

> 86 辛恩×蕾娜，从首图到成片的完整步骤说明，适用于 Seedance 2.0。  
> **提示词**：每段 `phases/phase01/phase01.txt`～`phases/phase12/phase12.txt` 内已有**可直接整段复制到 Seedance 2.0** 的**中文提示词**（含本段秒级动作·演出·运镜·场景细节与台词概述），无需再拼接总述。  
> **素材与衔接**：详见 `00-项目总述与素材说明.md`。

## 二、段 1：图 + 文生视频（仅此段用图生）

| 步骤 | 操作 |
|:----:|------|
| 1 | 在 Seedance 2.0 中选择 **图生视频**（或 Image + Text to Video）。 |
| 2 | **上传首图**：上传「首图」`section03_ref_img.jpg`（**不是**角色参照图；此图=场景+两人+露台傍晚夜景，仅段 1 使用）。 |
| 3 | **时长**：设为 **15 秒**。 |
| 4 | **提示词**：打开 `phases/phase01/phase01.txt`，将「以下整段复制到 Seedance 2.0 提示词框」下方的**整段中文**复制到 Seedance 提示词框即可（已含风格、角色、本段秒级动作·运镜·场景与台词）。 |
| 5 | **负面词**：在同一文件中复制「负面词」整段到 Seedance 负面词框。 |
| 6 | **比例**：原比例（或与首图一致）。 |
| 7 | 生成并下载视频，命名为如 `section03_seg01.mp4`。 |
| 8 | **截取尾帧**：必须截取该段视频的**最后一帧**，导出为 `section03_tail_01.png`，供段 2 作为 V2V 的视觉起点。 |
| 9 | **环境氛围参考图**：从段 1 视频或首图中选取一帧（露台、傍晚、城市灯光清晰），保存为 `refs/env_atmosphere/section03_env_ref.png`，供段 2～12 使用（若平台支持多图）。 |

## 三、段 2～12：V2V 接力（用上一段尾帧）

每一段都采用「**上一段尾帧图** + 总述 + 本段提示词」做 **Video-to-Video（V2V）** 生成。若平台支持多图，可同时上传**环境氛围参考图**以锁定露台傍晚光线与城市夜景。

| 步骤 | 操作 |
|:----:|------|
| 1 | 在 Seedance 2.0 中选择 **V2V**（或 Video-to-Video / 以图/视频为参考）。 |
| 2 | **上传参考**：上传 **上一段尾帧图**（段 2 用 `tail_01.png`，段 3 用 `tail_02.png`，…，段 12 用 `tail_11.png`）。若平台支持多图，可额外上传 `refs/env_atmosphere/section03_env_ref.png` 作为环境/打光参考。 |
| 3 | **时长**：**15 秒**。 |
| 4 | **V2V 强度**：建议 **30%～40%**，可先试 35%。 |
| 5 | **提示词**：打开当前段对应的 `phases/phaseXX/phaseXX.txt`，将「以下整段复制到 Seedance 2.0 提示词框」下方的**整段中文**复制到 Seedance 提示词框即可。 |
| 6 | **负面词**：在同一文件中复制「负面词」整段到 Seedance 负面词框。 |
| 7 | 生成并下载，命名如 `section03_seg02.mp4`～`section03_seg12.mp4`。 |
| 8 | **为下一段准备尾帧**：从本段视频截取**最后一帧**，保存为对应的 `section03_tail_02.png`～`section03_tail_11.png`（**段 12 无需再截**）。 |

**循环**：对段 3～12 重复上述「上传上一段尾帧图（+ 环境氛围参考图）→ 总述+本段提示词 → 生成 → 截尾帧」，直到 12 段全部生成完毕。

## 四、导出与剪辑成片

| 步骤 | 操作 |
|:----:|------|
| 1 | **按时间轴拼接**：在剪辑软件中，按 README「五、剪辑总时间轴」顺序导入 `section03_seg01.mp4`～`section03_seg12.mp4`，首尾相接，总长 3 分钟。 |
| 2 | **对齐配音**：若有日文配音，按每段 0～15s、15～30s、…、2:45～3:00 对齐到对应画面。 |
| 3 | **统一调色/音量**：可对 12 段做轻微色彩统一与音量平衡，使观感一致。 |
| 4 | **导出成片**：输出为最终 MP4。 |

## 五、常见问题与防崩脸提醒

| 问题 | 建议 |
|------|------|
| **脸型/画风突变** | 每段提示词中务必保留「consistent character」「Japanese anime style」及场景词；V2V 强度不要超过 40%。 |
| **动作跳跃** | 尾帧必须是**视频最后一帧**，才能保证下一段开头与上一段结尾连贯。 |
| **环境/光线漂移** | 使用「Same balcony evening lighting and city night view as segment 1」等锚定句，并尽量上传环境氛围参考图（段 2～12）。 |
| **时长不足 15s** | 确认 Seedance 2.0 当前版本单段上限为 15s，若平台限制更短，需在 README 与 00 中把每段时长改为实际最大值并相应增加段数。 |
