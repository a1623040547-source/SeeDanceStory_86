## 三、生成核心参数（API）

- **接口**：Seedance 2.0 API（https://seedanceapi.org/docs）

- **比例**：16:9 或与首图一致
- **提示词**：中文，**最大 2000 字**；每段仅 **1 张** image_url（首图或上一段尾帧 URL）
- **参考图 URL**：通过 `./scripts/upload-image-url.sh <图片路径>` 上传至 ImgBB 后取得
