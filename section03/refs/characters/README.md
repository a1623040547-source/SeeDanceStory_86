# 角色形象参照图

本文件夹用于存放**项目启动与生成时所需的角色形象图**。

## 放置内容

- **辛恩（Shin）** 单人或半身定妆图（86 原作动画风，表情/服装与剧本一致）
- **蕾娜（Lena）** 单人或半身定妆图（86 原作动画风）
- 可选：双人定妆图，便于统一两人比例与画风

## 使用方式

- 段 1 若采用「角色参照 + 场景图」分开时，将本文件夹内的角色图上传至 Seedance 的**角色参考**槽位。
- 若段 1 仅用一张首图（场景+两人），可将本文件夹中的图作为绘制/生成首图时的参照，或与 `refs/scenes/` 中的场景图合成后使用。

## 命名建议

- `shin_ref.png` / `lena_ref.png`
- 或 `section03_character_shin.png`、`section03_character_lena.png`
