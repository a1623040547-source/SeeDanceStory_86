# 《86｜不存在的日常》露台·傍晚小剧场 日文配音完整剧本

> 86 辛恩×蕾娜，露台傍晚看城市夜景，递外套与靠肩的温馨小段。**每 15s 内对话有信息量、有呼应或小转折**，避免一板一眼。总时长约 3 分钟，每段 15s，适配 Seedance 2.0 单段上限。

## 一、项目概述

| 项目 | 说明 |
|------|------|
| **风格** | 86 原作动画风 |

## 四、逐段完整剧本（含精确时间轴与日文配音）

### 段 1｜图 + 文生视频｜15s

**画面**：两人在露台并排靠栏杆，看城市夜景，傍晚天光，城市灯光点缀。蕾娜先开口，再追问辛恩。

**日文配音 · 蕾娜**

- 0～3s：（軽く息を吐く / 轻轻呼气）
- 4～8s：きれいだね……（Kirei da ne... / 好漂亮啊……）
- 9～12s：シンは？ 何、考えてる？（Shin wa? Nani, kangaeteru? / 辛恩你呢？在想什么？）
- 13～15s：（まばたき・夜景を見る / 眨眼、望向夜景）

**Seedance 提示词**  
见 `phases/phase01/phase01.txt`（含秒级动作·运镜·场景·中文提示词）。

### 段 2｜V2V 接力｜15s

**画面**：辛恩简短答「嗯、没什么」，蕾娜「哼—」侧头看他、嘴角微扬。

- 0～3s：……ああ。別に。（……Ā. Betsu ni. / ……嗯。没什么。）
**日文配音 · 蕾娜**

- 4～6s：ふーん。（Fūn. / 哼—。）
- 7～15s：（彼の横顔を見る・まばたき・少し笑う / 看他的侧脸、眨眼、微微笑）

**提示词**  
见 `phases/phase02/phase02.txt`。
