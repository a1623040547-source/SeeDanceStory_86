# phase08 提示词拆段（排查违禁词用）

复制到 Seedance 提示词框时，逐段粘贴测试，找出触发「不符合规范」的那一段。

**用法建议：**
- 先只粘贴 **01**，能通过则再试 **01 + 02**，再试 **01 + 02 + 03**……直到某次加上新的一段后报错，即该段含违禁或敏感词。
- 或反过来：先贴 **07（负面词）** 到负面词框，再在提示词框从 01 起逐段加。
- 负面词单独粘贴到平台的「负面词」输入框，其余 01～06 按顺序粘贴到「提示词」输入框。

| 文件 | 内容 |
|------|------|
| 01-项目风格规则.txt | 风格、段落、人物、配乐 |
| 02-场景与氛围.txt | 浅滩、浪花、氛围 |
| 03-景别与运镜.txt | 中景→近景、尾帧 |
| 04-位置关系.txt | 两人贴近、接吻、收束 |
| 05-本段动作与节奏.txt | 0～4s / 5～15s 时间块 |
| 06-本段台词概要.txt | 语气词、口型 |
| 07-负面词.txt | 负面词（贴到负面词框） |

排查出问题段后，可在该段内再拆句或换词精确定位，并把结论记入 `.cursor/skills/seedance-forbidden-words/forbidden-words.md`。
