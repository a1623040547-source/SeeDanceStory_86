# 服饰参考（refs/clothes）

## 用途

存放本 section 的**服饰参考图**，用于生成角色 ref、俯视图或提示词中的穿着一致性。本 section 为 S2 玩水·互动，服饰为**蕾娜泳装、辛恩沙滩装**。

## 本 section（S2）服饰

见 story/01-海边度假/00-故事总述与大纲.md 中「各 Section 服饰描述」：

- **辛恩**：与 S1 同款或同风格；短裤或卷起裤脚至小腿；赤脚或拖鞋；可略湿。
- **蕾娜**：泳装（S1 段 8 收束后换装、S2 段首即泳装）；可略湿、水珠。

## 放置内容与命名建议

- 辛恩 S2 服饰参考：`section02_clothes_shin.png` 或类似。
- 蕾娜 S2 服饰参考（泳装）：`section02_clothes_lena.png` 或类似。

同一 section 内各 phase 提示词与 ref 须与该 section 服饰一致（见《section-phase-script-retro》穿着服装状态）。

## 用法

- 制作 refs/characters 角色图时，可参照本目录服饰图保证穿着一致。
- 撰写或调整 phase 提示词时，若需强调服装细节，可引用本目录图片或 story 00 服饰描述。
