# 《86｜不存在的日常》海边度假 · 故事总述与大纲

> 86 辛恩×蕾娜，海边度假小长篇。**总时长约 10 分钟**，在 section 之上用 story 组织，为分镜切换提供更多选择。角色设定见 @.cursor/skills/protagonist-identity/docs/characters.md。

## 一、故事总述

| 项目 | 说明 |
|------|------|
| **标题** | 海边度假（暂定） |
| **主角** | 辛恩（Shin）× 蕾娜（Lena），与 protagonist-identity 一致 |
| **时间线** | **获得和平之后**的一次旅行（《86-不存在的战区》原著战后） |

### 本故事与原著·谈话中的原著要素

- **对话可自然融入《86-不存在的战区》原著情节与记忆**，点到为止、不喧宾夺主，以日常与情绪为主。
- **可用的原著向话题示例**（按 section 择一或少量使用，不必每段都提）：
  - **战争与和平**：能这样一起看海/旅行真好；偶尔提到「以前从没想过能这样」类的一句。
  - **战友与过去**：若提到 86 的同伴、军团、或「那时候」的回忆，用简短一句或蕾娜问、辛恩简短答即可（如「大家要是也能来就好了」「……嗯。下次吧。」）。
  - **未来与结婚**：在 S4 傍晚或 S5 收尾可有一两句对婚礼/未来的轻语或默契（不必直白说「结婚」，可用「之后」「以后」或眼神/微笑带过）。
- **原则**：以海边度假的轻松与二人互动为主；原著梗作为**底色与共鸣**，不变成回忆杀或沉重独白。台词保持与 protagonist-identity 一致（辛恩简短、温柔带笑意；蕾娜有起伏、可调侃可安静）。

## 二、剧情弧与 Section 划分

全篇按**时间/地点/情绪**拆成 5 个 section，每个约 2～2.5 分钟（约 8～10 个 phase）。节奏上**快段与慢段交替**，寡言方（辛恩）每段均有对蕾娜的**可执行微反应**（视线、喉结、眉/嘴角、手指等），蕾娜先开口、有调侃与情绪起伏。

| Section | 标题（暂定） | 约时长 | 场景 | 主要事件与情绪 | 节奏要点 |
|---------|--------------|--------|------|----------------|----------|
| **S1** | 抵达·民宿/海边小镇 | ~2 min | 民宿玄关/房间·窗外见海 | 入住、放行李；蕾娜兴奋张望、辛恩沉稳拎包跟随（温柔带笑）；简短对话「到了哦」「嗯」；蕾娜提议去海边；可选一句「和平后第一次这样的旅行」类 | 中速；蕾娜话多、辛恩简短接话+视线/微反应 |
| **S2** | 玩水·互动 | ~2 min | 浅滩/水边 | **肢体互动突出亲密**：踩水、泼水玩笑（蕾娜撩水泼他、辛恩躲或挡、嘴角微动）；**可设计**：牵手或挽臂踩水、蕾娜滑一下时辛恩拉一把/轻扶腰或肩、泼水嬉闹中身体靠近或从背后扶她防滑；递毛巾时手指触碰、或辛恩帮她拨开沾在脸上的头发/水珠；蕾娜「别总板着脸嘛」、辛恩简短回一句或反手轻撩水回应；肩膀相碰、她不躲他也不躲，自然贴近平稳收束。 | 快段集中；短句/接话/语气词；触发-反应链明显；肢体接触可执行、自然不刻意 |
| **S3** | 海边散步·日间 | ~2.5 min | 沙滩·浪花·日照 | 两人沿沙滩走；蕾娜脱鞋踩沙、辛恩看她；蕾娜说「好舒服」、追问辛恩感受；**可融入原著**：如「能这样走着真好」「以前从没想过」等一句；可有一段留白（脚步、海浪、视线） | 中→慢；至少 1 段留白（少台词、动作与神态为主） |
| **S4** | 傍晚·沙滩或露台 | ~2 min | 沙滩/露台·日落 | 日落、光线转暖；递外套/靠肩或并肩坐；安静对话、微反应填满；**可融入原著/结婚**：对「之后」「以后」或婚礼的轻语、默契一笑（不必直白，用眼神/一句带过） | 慢段为主；留白、视线与呼吸；环境氛围统一（暖色、傍晚） |
| **S5** | 收尾·民宿床上私语 | ~1.5 min | 民宿·床上 | 两人一起躺在民宿床上私语；安静、亲密；可选一句对未来的轻语或默契；极简收束，镜头可拉远或留白结束 | 慢段；台词少而轻；私密、收束 |

**全篇快段/慢段分布**（与《section-phase-script-retro》台词与节奏一致）：

- **快段**：S2 玩水（多句短台词、接话、调侃）。
- **慢段**：S3 中段散步留白、S4 傍晚安静、S5 收尾；S1 中速。
- 各 section 内 phase 的台词句数/字数需有差异，避免每段均分两句。

## 三、Section 间过渡与分镜选择

为分镜切换提供可选方案，便于剪辑与观感连贯。

**过渡责任的落点**：section 间过渡的**实际任务落在每个 section 的第一个 phase（段首）与最后一个 phase（段尾）**。撰写各 section 时须在以下位置落实：
- **段尾 phase**：在情节说明与提示词中写清本段**收束方式**（动作结束/景别/情绪落点），为跨 section 过渡预留；避免断在动作或台词半途。
- **段首 phase**：在情节说明与提示词中写清**承接上一 section**、本 section 开场（场景/光线/情绪）。
剪辑时的黑场、淡入淡出等与段尾/段首画面配合即可。

**构建剧本时 Section 尾部收束**：每个 section 的**尾部**（尤其是最后一个 phase）应是**收束的**——该 section 内情节与情绪有明确收束点（如动作结束、视线落点、静帧感、空镜），便于本 section 独立观看，也便于与下一 section 的过渡。

| 过渡 | 推荐方案 | 剪辑/备注 |
|------|----------|-----------|
| **S1 → S2** | 淡出淡入 或 黑场 0.3～0.5s + 「海边」空镜 1～2s 再进人 | 从室内到户外，时间可连续；若用黑场则暗示「稍后」 |
| **S2 → S3** | 首尾帧 V2V 连续 或 动作切点（如从水边走上沙滩，段尾离水→下段从散步起） | 保持日照与服装一致 |
| **S3 → S4** | **时间跳跃**：黑场 0.5s + 字幕或空镜「傍晚」再进人；或淡出淡入 | 光线/色调从日间转傍晚，需在 S4 首段提示词中明确傍晚氛围 |
| **S4 → S5** | **场景切换**：淡出淡入 或 黑场 + 空镜（海/夜）后切回民宿室内；从傍晚户外到民宿床上，时间可暗示「当晚」 | 情绪从「安静相处」到「床上私语」，节奏放缓、私密收束 |

**可选**：若某 section 最后一 phase 以**景别收束**（如拉远、空镜），下一 section 可从新景别/新机位开场，在各自 00 或 README 的 phase 衔接总表中标注。

## 四、环境·氛围·穿着统筹

- **场景**：海边小镇、民宿（S1、S5 床上）、沙滩·日间（S2、S3）、沙滩/露台·傍晚（S4）。全篇**第一段环境氛围参考图**建议取自 S1 或 S2 首段（室内暖光或海边日照），后续 section 在 00 中约定「与 story 氛围参考一致」或按 section 切换（傍晚段单独约定暖色、低对比）。
- **光线**：S1 室内自然光；S2～S3 日照、高亮与阴影明确；S4 傍晚暖色、逆光/侧光可；S5 民宿室内夜灯或微光，私密柔和。
- **穿着**：
  - S1：便服（如夏装、蕾娜连衣裙/短袖，辛恩 T 恤/衬衫）；**S1 段尾辛恩提议蕾娜穿泳装、蕾娜去换装（无换装画面），跳转 S2 时蕾娜以泳装登场**。
  - S2～S3：**蕾娜泳装**（S1 段 8 收束后换装、S2 段首即泳装，先玩水、再散步）；辛恩便服或沙滩装（拖鞋、卷裤脚/短裤）；**若有脱鞋/卷袖等变化，该 section 内各 phase 提示词与 ref 须一致**（见《section-phase-script-retro》穿着服装状态）。
  - S4：可加薄外套/披肩（递外套情节时，脱外套后各 phase 显式写「无外套」或备 ref）。
  - S5：**民宿床上**；换回便服或睡衣风（与 S1 风格一致或略宽松），私密场合、两人躺床私语。
- **配乐**：若某 phase 使用 BGM，须在提示词中写明「配乐在 15s 内封闭、结尾前有明确渐进终止，且**最后 1 秒不能有音乐**」；否则尽量不用配乐（见《section-phase-script-retro》配乐与 15s 封闭）。

## 五、复盘要点在本 story 的落实

| 复盘要点 | 在本故事中的落实 |
|----------|------------------|
| **寡言/内敛人设与微反应** | 辛恩每段均有对蕾娜的微反应（视线、喉结、眉/嘴角、手指、不躲开靠近）；section 的 00/README 写明「寡言≠无反应」，phase 的【本段动作与节奏】写清「蕾娜做 X→辛恩做 Y」的触发-反应链。 |
| **快段/慢段与台词密度** | S2 为快段（多句短台词）；S3 中段、S4、S5 含留白段；各 section 内 phase 台词句数/字数有差异。 |
| **环环相扣** | 每个 phase 的【本段情节说明】含「承接上段…」与「本段以…结束、下段将…」；提示词若允许可带一句前后衔接。 |
| **Phase 间与 section 间镜头过渡** | Phase 间：各 section 在 00 或 README 做 phase 衔接总表（首尾帧/动作切点/景别）。Section 间：见上文「Section 间过渡与分镜选择」。 |
| **环境/氛围/打光** | 每 section 约定与 story 一致或本 section 统一；S4 傍晚单独约定暖色；第一段环境氛围参考图在 S1 或 S2 首段确定，段 2～N 可复用。 |
| **穿着状态跨 phase/section** | 脱外套/脱鞋/卷袖等变化后，后续 phase 提示词与 ref 显式写穿着状态；跨 section 时在 00 中注明「S4 穿着：蕾娜披外套、辛恩无外套」等。 |
| **全能参考 15s** | 每 section 若用全能参考，拆成 7s（环境）+ 8s（连续性）两个独立视频，不拼接。 |
| **配乐 15s 封闭** | 使用配乐的 phase 在提示词中写明配乐须在 15s 内封闭、结尾前渐进终止、**最后 1 秒不能有音乐**；否则尽量不用。 |
| **门与场景描述**（中期复盘） | 环境中**门一律为自动门（横向滑动进出）**，不写「用手开门」——Seedance 无明确提示时人物难以正确执行开门动作。**环境设计须细致**：像小说家一样把场景细节交代清楚（空间布局、家具位置与样式、材质光线、动线），refs/scenes 与各 section 00 中应有完整场景描述，而非一句笼统概括。见《section-phase-script-retro》docs/16-环境与门的设计。 |
