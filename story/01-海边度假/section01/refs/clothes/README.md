# 服饰参考（refs/clothes）

## 用途

存放本 section 的**服饰参考图**，用于生成角色 ref、俯视图或提示词中的穿着一致性。可与 refs/characters 配合：characters 为角色脸/身形，clothes 为本 section 服装细节（如 S1 便服的上衣、下装、鞋等）。

## 本 section（S1）服饰

见 story/01-海边度假/00-故事总述与大纲.md 中「各 Section 服饰描述」：

- **辛恩**：夏装便服，深色或军装风 T 恤/短袖衬衫、长裤或九分裤、休闲鞋；无外套。
- **蕾娜**：夏装便服，浅色连衣裙或短袖+半裙/短裤、凉鞋或平底鞋；发型可马尾或披肩。

## 放置内容与命名建议

- 辛恩 S1 服饰参考（可俯视图或全身）：`section01_clothes_shin.png` 或类似。
- 蕾娜 S1 服饰参考：`section01_clothes_lena.png` 或类似。
- 双人服饰参考（可选）：`section01_clothes_both.png`。

可用于生成人物俯视图/全身 ref 时与 story 00 的「服饰描述」一致；同一 section 内各 phase 提示词与 ref 须与该 section 服饰一致。

## 用法

- 制作 refs/characters 角色图时，可参照本目录服饰图保证穿着一致。
- 撰写或调整 phase 提示词时，若需强调服装细节，可引用本目录图片或 story 00 服饰描述。
