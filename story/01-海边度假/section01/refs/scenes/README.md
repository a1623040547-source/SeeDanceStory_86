# 场景图 / 段 1 首图（refs/scenes）

## 用途

存放**段 1 图生视频**使用的场景图或首图。段 1 为「两人刚进民宿玄关，窗外见海」的镜头，本目录图片将作为图生的视觉起点。

---

## 设计约束（Seedance 适配）

- **门**：本 section 中所有涉及「门」的进出，在描述与 phase 提示词中**一律使用自动门（横向滑动进出）**。人物不执行「用手开门」（不写推门、拉门、转门把）；写「自动门横向滑开、人物走出/进入」等。详见《section-phase-script-retro》docs/16-环境与门的设计。

## S1 场景细致描述（供提示词与图生一致）

以下为**小说家级别的场景设定**，撰写 phase 提示词【场景与氛围】、制作或选用本目录场景图时，应与此一致，保证各段画面中空间、家具、光线、人物锚点统一。

### 整体

- 海边小镇的一栋**小型民宿**，白墙或浅米色外墙，可能带木饰或浅色窗框；入口朝街或朝庭院，**入口为自动门（横向滑动）**，门框与墙同色或深色细框，玻璃门扇横向开合，无人手触门。
- 室内为**玄关与主室一体**的开放空间，无单独走廊；从自动门进入即玄关，向前数步即进入同一房间，房间内侧为窗。动线：**自动门（画右或身后）→ 玄关 → 室内 → 窗（画左或正面）**。

### 玄关（进门处）

- **门**：**自动门**，横向滑动开闭；门框清晰，玻璃或半透明，可略见门外光；人物「进门」时写「自动门横向滑开、两人步入」，「出门」时写「两人走向自动门、门横向滑开、走出」。
- **地面**：木质地板，浅色（原木色或淡黄褐），与室内地板连续；玄关处可略设一块深色或同色地垫，不抢戏。
- **墙**：与室内统一，白或米白、干净；门侧墙可挂一枚小挂钩或简约置物板，无杂乱。
- **家具**：门内侧**右手边**（以进门面向室内为基准）靠墙有一**矮柜**，齐腰高，木质或白色，台面可放钥匙盘或小盆栽；**行李在本 section 中于 phase02 放在此矮柜旁或台面旁**，之后不再移动。玄关与室内之间无隔断，仅以「靠近门」与「靠近窗」区分。

### 室内（玄关至窗之间）

- **窗**：在**与门相对的一侧**（即房间深处或画中/画左），一整面或大半面窗，**落地或接近落地**，窗框为白或浅木色；**窗外所见**：海平线在画面中上部，天空占多半，可有一线远山或云；无阳台遮挡时海与天直接填满窗景；自然光从窗照入，为室内主光源，午后偏暖、明亮。
- **地板**：与玄关同款木地板，整室一致；无地毯时更显简洁。
- **墙**：白或米白，窗侧墙即窗框所在；无复杂装饰，可有小幅画或留白。
- **家具（与 00 空间布局一致）**  
  - **矮柜**：已在玄关门侧说明；若剧本中「放帽子/包」的位置在门侧，即此柜台面或柜旁；若在「沙发旁」，见下。  
  - **衣架**：靠窗侧墙或靠玄关侧墙**一具**，细高、木质或金属，可挂外套或包；不挡窗、不挡动线。  
  - **沙发**（可选）：一张双人小沙发或长沙发一侧，靠窗或靠侧墙，浅灰或米色布艺；若设定有「蕾娜拿帽子/包」从沙发旁取，则小几或沙发旁有一处可放帽、小包。  
  - **小桌/几**（可选）：若沙发存在，则沙发前或侧有一小几；否则室内可仅矮柜+衣架，极简。  
- **行李**：仅在 phase01～02 出现；phase02 结束时**已放在门侧矮柜旁（或矮柜与墙之间）**，phase03～08 不再写「放行李」，镜头中行李可始终处于该位置。

### 光线与氛围

- **主光源**：来自窗的自然光，从房间深处（窗侧）照向玄关方向；玄关因背窗略暗于窗边，但整体仍为**暖色、明亮**，无阴冷感。
- **时间感**：午后抵达，夏日或夏末，窗外天空晴朗、海面平静；室内**安静、抵达后的松弛感**，86 动画风、治愈、不压抑。

### 空间锚点（写 phase 提示词时用）

| 锚点     | 位置说明 |
|----------|----------|
| 自动门   | 玄关入口，画右或两人身后；横向滑动，不写用手开门。 |
| 矮柜     | 门内侧右手边靠墙；行李在 phase02 后固定放于此旁。 |
| 窗       | 与门相对一侧，画左或画中背景；窗外海平线+天空。 |
| 衣架     | 靠窗侧墙或玄关侧墙，一处。 |
| 沙发/帽·包 | 若有沙发，则「拿帽子/包」在沙发旁或矮柜旁，与 00 物品时间线一致。 |

### 动线摘要

- **进门**：自动门横向滑开 → 两人步入玄关（蕾娜先、辛恩拎行李在后）。
- **玄关→窗边**：少女向窗方向走，少年在门侧放行李于矮柜旁 → 少女到窗边扶窗望海，少年在室内望向她。
- **窗边→出门**：两人从窗边/室内向玄关移动 → 走向自动门 → **门横向滑开、两人走出**（不写「推门」「开门」）。

## 环境描述（用于生成环境图·精简句）

供图生或手绘时一句锚定氛围用；**细节以本节「S1 场景细致描述」为准**。

- **室内一隅**：海边民宿，玄关与房间一体；**自动门在画右或身后（横向滑动）**；窗在画左或正面，落地或近落地，窗外海平线与天空、自然光暖色；木质地板、白或米白墙；门侧矮柜、衣架一处、可选小沙发与小几；86 动画风、暖色、安静、抵达感。
