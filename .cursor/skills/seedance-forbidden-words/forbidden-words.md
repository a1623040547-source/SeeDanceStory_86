# Seedance 2.0 违禁词与替换建议

以下为「复制到即梦 Seedance 2.0 提示词框 / 负面词框」时需避免或替换的用词。依据实际生成通过情况整理，非官方完整名单。

---

## 一、版权 / IP 类（易触发内容审核）

| 类型       | 避免使用示例              | 替换建议 |
|------------|---------------------------|----------|
| **作品名** | 86, Eighty Six            | Japanese anime style / anime style consistent with reference image。**实测更可能触发违禁的是作品名。** |
| 角色名     | Shin, Lena（辛恩、蕾娜等） | 在「本段情节说明」「本段素材」「剧本台词」等**不复制到 Seedance 提示词框**的段落中使用角色名可保留。**复制进 Seedance 的提示词块内**（【项目·风格·规则】～【负面词】）不用真实角色名，改用**人称代指**（见下方「人称代指规范」），可避免「不符合规范」。 |

**原则**：复制进 Seedance 的**提示词**（无论中英文）中均建议避免作品名（如 86、Eighty Six、《86｜不存在的日常》等），易触发「输入的文本不符合规范」；可改为「日式动画风，与角色参考图一致」。角色名（辛恩、蕾娜）在情节说明、台词块等不复制到提示词框的段落中可保留。

**人称代指规范**（仅用于复制到 Seedance 的提示词块【项目·风格·规则】～【负面词】内，避免角色名触发审核且代指明确）：
- **仅两人时**：用 **男主、女主** 指代（实测「少年、少女」整份粘贴时可能触发「不符合规范」）；对称、明确，如「男主沉稳少言、女主活泼先开口」。
- **三人及以上时**：用 **男一、女一** 或 **男主、女主** 指代主要角色；第三人起用 **男二、女二** 或 **配角（可加简短外貌/身份）**，例如「男一与女一并排，男二在侧后方」。同一 phase 内代指须一致。

## 五、整份粘贴时的语言（实测结论）

**现象**：phase07 整份复制到平台时报「不符合规范」，平台不提示具体位置。经逐一尝试（改情节说明、改台词、改英文用词、去掉日文配音词）仍不合规，**将提示词与负面词全部改为中文后通过**。

**结论**：整份粘贴时，**英文的提示词或负面词**可能触发审核，但无法确定是哪一个词（可能是某几个英文词，也可能是整段英文被统一处理）。平台未提供具体违禁词反馈。

**建议**：
- 若你习惯**整份复制** phaseXX.txt 到平台：将该 phase 的「提示词框」和「负面词」都写成**中文**即可通过；**日文配音词（【配音词：…】）可以保留**在文件内，实测不会触发不合规。
- 若你**只复制提示词 / 负面词**到对应输入框：可继续用英文，但避免本表已列出的词（glares、annoyed、作品名等）。
- **结论**：导致不合规的是英文的提示词/负面词，不是日文配音词。

## 六、其他（按需补充）

- 暴力、色情、敏感政治等：通用内容政策，不写入提示词。
- 真人名人：即梦对真人脸有单独限制，与提示词违禁词分开。
- **整份文件粘贴**：若将 phaseXX.txt 整份复制到平台，上述所有块（情节说明、台词、提示词、负面词）均可能被扫描。
- **英文 phase + 数字/范围**：在会被复制到平台的段落（含【位置关系·物品关系】等）中，避免写「phase03～08」「phase02」等，易触发「不符合规范」。改用中文：**第 N 段**、**第 N～M 段**（如「第3～8段」「下段（第2段）」）。
- **section08 提示词块实测（结论）**：**接吻、接吻画面**可用，非违禁。实测触发「不符合规范」的为：① **section**（英文）→ 改用「段落」；② **15s、1s、0～4s、5～15s**（英文单位 s）→ 改用「15 秒、1 秒、0～4 秒、5～15 秒」；③ **闭眼或微迎上** → 改用「微仰脸迎上」；④ **「嗯」「唔」**（提示词框内带引号语气词）→ 改用「轻声语气词、气声」或「轻声『嗯』『唔』」等。后续写吻戏或亲密段提示词时保留「接吻」即可，其余按上替换。
- **暧昧/氛围类词汇实测（批次1）**：以下词单独成批粘贴**通过**：暧昧、情欲、诱惑、撩人、性感、欲拒还迎、缠绵、温存。提示词中可酌情使用，此前 section08 将「暧昧」改为「温馨」可能为组合触发或误判。
- **亲吻/肢体接触类词汇实测（批次2）**：以下词单独成批粘贴**通过**：接吻、亲吻、唇瓣相触、轻吻、深吻、拥抱、搂抱、依偎、贴紧、身体贴近。吻戏或亲密段提示词可放心使用上述表述。
- **身体部位（含蓄）实测（批次3）**：以下词单独成批粘贴**通过**：腰、颈、肩、胸口、身前、指尖、手搭在腰侧、轻扶后背。提示词中可放心使用上述部位与动作表述。
- **身体部位（较直白）实测（批次4）**：胸部、臀部、大腿、肌肤、若隐若现、曲线**通过**。**裸露** 触发「不符合规范」，避免使用；可改用「若隐若现」「肌肤」「曲线」等含蓄表述。
- **动作/状态类实测（批次5）**：以下词单独成批粘贴**通过**：脱、解开、抚摸、触碰、呼吸急促、闭眼、半阖眼、迎上。提示词中可放心使用；此前 section08 将「闭眼或微迎上」改为「微仰脸迎上」可能为组合或上下文触发。
- **单字/短词实测（批次6）**：吻、唇、胸、臀、欲、色、床、睡**通过**。**裸** 触发「不符合规范」，避免使用（与「裸露」一致）；可改用「肌肤」「若隐若现」等。
- **泳装、沙滩装**：实测触发「不符合规范」。改用 **服装与参考图一致** 或 **男主与女主、服装与参考图一致**，不写具体服装类型。
- **少年、少女（整份粘贴时）**：整份复制时可能触发。改用 **男主、女主**。
- **标题与正文中的英文**：如 phase08、Section、V2V、15s、section02_tail 等易触发。标题改用「第8段、上一段尾帧衔接、15秒、段落末·收束」；时间用「秒」；素材说明用「上一段尾帧、本段尾帧」等。
- **新增条目**：确认某词导致不通过后，在本文件对应分类下新增一行，格式同上，便于后续 phase 统一改写。

**接吻/亲吻类（对应六·批次2）**

自然贴近、站定贴近、指尖、手搭肩或胸前、腰侧或颈侧、掌心贴、指尖微收、轻扶、指尖可轻攥衣料、手搭肩臂、依偎贴紧、头微侧。

**神态/动作类（对应六·批次5）**  
闭眼、半阖、闭眼或半阖、睫毛微颤、眉心由绷渐松、唇线微变、眉宇放松、嘴角极轻、视线落她唇、低头靠近、微仰脸迎上（若需更保守可替「闭眼或微阖迎上」）。

**语气/声音**  
无完整台词、语气词、气声、呼吸、轻声嗯唔、偶有嗯唔气声、轻叹、喉音或气声。

**画面约束**  
仅两人、无其他人物、同场景同角色；尾帧为接吻画面、段尾以接吻画面结尾。
