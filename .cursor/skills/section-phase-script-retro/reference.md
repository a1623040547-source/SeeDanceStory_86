# 剧本复盘 · 对比示例（单薄 vs 改进）

供撰写或改写 phase 时对照：左为「易导致活动性低/节奏平」的写法，右为改进方向。

---

## 1. 台词密度与节奏

### 单薄示例（每段两句、长度接近）

```
段3：0～7s 一句，8～15s 一句
段4：0～5s 一句，6～12s 一句，13～15s 一句
段7：0～8s 一句，9～15s 一句
→ 整篇节奏均匀，没有「快段」与「慢段」对比。
```

### 改进示例

**快段（如抢零食/拌嘴）**  
- 时间块拆细：0～3s / 3～6s / 6～9s / 9～12s / 12～15s。  
- 短句、接话、语气词、轻笑：「哎」「那块是我的」「才不是」「じゃあ、これもらう」「クスッ」等。  
- 剧本台词块里体现多句、多时间点。

**慢段（如递茶/盖毯子/收尾）**  
- 大块留白：0～10s 仅动作与神态（递茶、接杯、微笑、盖毯子），10～15s 一句收束或无声。  
- 剧本台词块里仅 1～2 句，或「（无台词，仅动作）」。

## 2. 动作 / 神态 / 细节描写

### 单薄示例（概括句）

**情节说明**  
- 「辛恩故作严肃抱起胳膊、说「太过了啊」并默默瞥一眼，蕾娜憋笑偷看；表面较劲实则快要笑场。」

**提示词本段**  
- 「Young man crosses arms with stern expression, young woman holding back laughter and glancing at him, playful mood.」

→ 动作与神态只有「抱臂」「严肃」「憋笑」「偷看」，缺少顺序与变化，模型难以生成细腻表演。

### 改进示例

**情节说明（补动作顺序 + 视线/微表情）**  
- 「辛恩先抱起胳膊、板起脸说「太过了啊」，说完视线扫向蕾娜、停在她脸上；蕾娜先抿嘴憋笑、视线瞟向他，再忍不住嘴角上扬、用手背挡一下嘴，又抬眼偷看他反应，表面较劲实则快要笑场。」

**提示词本段（可执行动作链 + A→B 互动）**  
- 中文示例：「本段：同一客厅。青年先抱臂、板脸说太过了啊，说完目光扫向少女并停住；少女先抿嘴憋笑、瞟向他，再嘴角上扬、手背轻挡嘴，又抬眼偷看他；青年保持板脸但嘴角微动。角色一致。台词：青年板脸说太过了然后沉默看她。」  
- 英文可类比：young man crosses arms first, then says it is too much with stern face, then gaze sweeps to young woman and holds; young woman bites lip to hold laugh, glances at him, then corner of mouth lifts and she covers with back of hand, then looks up at him again; young man keeps stern look but corner of mouth twitches. Dialogue: young man says it is too much then looks at her in silence.

**要点**  
- 谁**先**谁**后**（先抱臂再说话再视线扫过去）。  
- **视线**（扫向、停住、瞟、抬眼）。  
- **微表情变化**（抿嘴 → 嘴角上扬、板脸但嘴角微动）。  
- **互动链**（他说完看她 → 她憋笑瞟他 → 再挡嘴抬眼看他）。

### 2.1 寡言/内敛人设与可执行微反应（section03 中期复盘）

**问题**：人设「沉稳、寡言」在成片中易被模型解读为「少动/无表情」，导致双人戏中该角色与对手方无触发-反应链，观感为单方活跃、另一方静态。

**单薄示例（秒级表·青年列）**

| 时间 | 青年（辛恩） | 少女（蕾娜） |
|------|--------------|--------------|
| 0–2s | 保持望夜景 | 头转向青年，说「シンは」 |
| 2–5s | 同 0–2s | 说「いつもこうだね」语气调侃 |
| 5–8s | 同 0–2s | 说完笑意明显，目光看他 |
| … | 保持/同左/静止 | … |

→ 青年列无对少女动作/台词的可执行回应，模型易生成「木头人」式表演。

**改进示例（触发-反应链）**

| 时间 | 青年（辛恩） | 少女（蕾娜） |
|------|--------------|--------------|
| 0–2s | 保持望夜景，耳根微侧（在听） | 头转向青年约 25 度，说「シンは」 |
| 2–5s | 察觉被看，眼珠向她方向动约 5 度仍不转头；喉结微动 | 说「いつもこうだね」语气调侃 |
| 5–8s | 视线仍朝前，眉极轻微动；呼吸一次、肩微动 | 说完笑意明显，目光落他侧脸 |
| … | 视线向她偏多约 10 度；像要开口又没问 | … |

**提示词【本段动作与节奏】改进**  
- 单薄：「少女头转向青年说辛恩你总是这样呢，青年望夜景。少女说什么都不说就做了笑意明显。」  
- 改进：「少女头转向青年说辛恩你总是这样呢，**青年眼珠微向她动、喉结微动仍不转头**。少女说什么都不说就做了笑意明显，**青年视线朝前眉极轻微动、呼吸一次肩微动**。」

**要点**  
- **人设与可执行动作区分**：寡言 = 少台词，≠ 无微反应；每一时间块该角色须有对对手方的**具体可执行反应**（视线/眼珠、喉结/眉/嘴角、呼吸/肩/手指）。  
- **触发-反应链**：对手做 X 的时段内，显式写出该角色的 Y（A 做 X → B 做 Y），避免仅「保持」「同左」「静止」。

## 3. 整段对比小结

| 维度 | 单薄 | 改进 |
|------|------|------|
| 台词节奏 | 每段 2 块、长度接近 | 有快段（多句/多块）与慢段（少台词/留白） |
| 情节说明 | 一句概括谁做了什么 | 动作顺序 + 视线 + 微表情变化 |
| 提示词本段 | 一句概括两人状态 | 动作链 + A→B 反应 + 至少一处表情变化 |
| 寡言/内敛角色（section03） | 秒级表该角色列多为「保持」「同左」「静止」；提示词只写对手方动作 | 该角色列每时间块有对对手方的可执行微反应；提示词显式写「对手做 X→该角色做 Y」触发-反应链 |

撰写新 phase 或改写 section 时，可直接对照上表与示例增补。

## 4. phase 间承接与镜头过渡方案

以 15s 为一 phase **构建剧本时**，除首尾帧衔接外，**在创建剧本阶段就可以提出**多种镜头过渡方案并写入 00/README 与 phase 边界，避免只依赖一种衔接方式；生成与剪辑阶段按剧本标注执行即可。

### 在 00 或 README 中的「phase 衔接总表」示例

| 衔接 | 过渡方案 | 剧本落点示例 |
|------|----------|--------------|
| 1→2 | 首尾帧 | 段 1 尾帧 = 段 2 首帧；段 2 提示词 same scene |
| 2→3 | 动作切点 | 段 2 尾「脚步渐远」；段 3 头「端茶盘入画、放茶几」 |
| 5→6 | 节奏切点 | 段 5 尾「两人对峙」；段 6 头「蕾娜整盘端走」；00 标注「5→6 节奏由紧到松」 |
| 7→8 | 黑场可选 | README 剪辑轴：「段 7 与 8 之间可加 0.5s 黑场」；段 8 情节说明写「破功笑场后」 |
| 10→11 | 景别变化 | 段 10 尾「中景」；段 11 头「递毯子特写/手部」；提示词本段写清景别 |

### 在单个 phase 内的边界标注示例

**情节说明**  
- 段尾：「本段以辛恩放下茶杯、手离杯为结束动作，供段 N+1 从下一动作续接。」  
- 段头：「本段从蕾娜接过茶杯、双手捧杯微笑开始，与段 N 尾帧（辛恩递出杯）衔接。」

**提示词**  
- 段尾可加一句（若平台允许多语言）：End with young man’s hand leaving the cup on the table.  
- 段头：Start with young woman receiving the cup, both hands around it, small smile.

### 与剪辑的配合

- 仅首尾帧 V2V：生成阶段只保证视觉连续；黑场、淡入淡出、景别切换在剪辑时加。
- 剧本先标好「此处可加黑场」「段 N 以特写开场」，成片时按表执行，避免临时硬切。

## 5. 当前 phase 内提及前后段（环环相扣）

为保证剧情与人物衔接自然，当前 phase 应**部分提及**上一 phase 尾部与下一 phase 首部，形成环环相扣。

### 单薄示例（只写本段）

**段 7 情节说明**  
- 「辛恩故作严肃抱起胳膊、说「太过了啊」并默默瞥一眼，蕾娜憋笑偷看；表面较劲实则快要笑场。」

→ 没有点明「上段末尾蕾娜整盘端走、辛恩愣住」的承接，也没有点明「本段以板脸瞥她结束、下段蕾娜破功笑场」的过渡，读起来像孤立的 15s，与前后段关系靠读者自己猜。

### 改进示例（环环相扣）

**段 7 情节说明**  
- 「承接上段蕾娜整盘端走后辛恩愣住。本段：辛恩故作严肃抱起胳膊、说「太过了啊」并默默瞥一眼，蕾娜憋笑偷看；表面较劲实则快要笑场。本段以辛恩板脸瞥她、蕾娜憋笑偷看结束，下段将破功笑场、两人和解。」

**段 7 提示词（段头/段尾可带一句衔接，若平台允许）**  
- 段头：承接上段少女将整盘零食挪向自己、青年愣住后。本段：青年抱臂…  
- 段尾：本段以青年板脸沉默看她、少女憋笑结束；下段由少女笑场、两人和解开场。

**要点**  
- **段头**：谁在什么状态/动作之后（承接上段尾部）。  
- **段尾**：本段收在什么画面/情绪，下段将从什么画面/情绪开始。  
- 不必每句都写，但每段情节说明里至少各有一处「承接上段…」与「本段以…/下段将…」。

## 6. 环境/氛围/打光统筹（section02 后期复盘）

**问题**：phase 往后生成时，虽保证了尾帧衔接，但环境光、窗外天气、整体氛围发生漂移（前段暖光+下雨，后段变下雪/冷光、画面变花）。

**改进**：

1. **refs 增加「环境氛围参考图」**  
   - 从段 1 视频或首图中选取一帧（暖光、雨窗、室内），保存为 `refs/env_atmosphere/section02_env_ref.png`。  
   - 段 2～12 生成时除上传尾帧外，额外上传该图作为环境/打光参考。

2. **提示词中锚定环境**
   - 在总述或每段提示词中明确：`Same warm indoor lighting and rainy window as segment 1; no snow, no cold blue tint.`
   - 与本 section 的「环境概述」一致（暖色室内光、窗外冷灰雨天光），避免后段漂移。

## 7. 运镜 / 镜头 / 程度说明（即梦向）

即梦等视频生成平台建议：**主体动态 + 运镜类型（速度/角度）+ 场景变化 + 风格**；支持推/拉/跟拍、景别控制等，提示词中写清运镜与程度更易出片可控。

### 单薄示例（无景别、无运镜类型与程度）

**段 12 提示词（原）**  
- 「青年与少女安静并排坐，一人喝茶一人看书，雨窗背景，平和收尾镜头稳定治愈氛围。」

→ 只有「镜头稳定」，没有景别（中景？近景？）、没有运镜类型与速度/幅度，平台难以精确执行。

### 改进示例（景别 + 运镜类型 + 程度）

**段 12 提示词（改进）**  
- 「**固定机位，中景**，双人半身，青年与少女安静并排坐于沙发，一人喝茶一人看书，雨窗在背景；**无运镜，画面保持稳定**，治愈氛围。」

**段 13 提示词（改进）**  
- 「**开场近景**：少女放下茶杯靠向青年、头枕其腿，青年轻抚少女头发。**运镜**：从近景**缓慢拉远**，约 5～8 秒内拉至**室内全景**（沙发、两人、雨窗、暖光），再切至**屋外远景**（雨中建筑/雨窗外观）。**程度**：拉远匀速、缓慢，无晃动。」

### 可复用用语速查

| 类型 | 建议用语（可直写进提示词） |
|------|----------------------------|
| 景别 | 特写（面部/手部）、近景（胸部以上）、中景（半身/膝上）、全景（全身）、远景（环境） |
| 固定 | 固定机位、无运镜、static camera、画面保持稳定 |
| 推 | 缓慢推进、轻微推进、从全景推至中景 |
| 拉 | 缓慢拉远、从近景拉至室内全景、逐渐拉远 |
| 跟 | 轻微跟拍、肩部跟拍（适合互动） |
| 程度 | 缓慢、轻微、微微、逐渐、匀速、约 N 秒内完成 |
