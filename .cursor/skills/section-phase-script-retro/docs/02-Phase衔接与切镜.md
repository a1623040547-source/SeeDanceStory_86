# 二、Phase 衔接与切镜（段间结构与时长）

本主题合并：**镜头过渡方案**、**环环相扣**、**V2V 切镜边界与 phase 时长**。用于自检「段与段怎么接、何时断、每段多长」。

---

## 1. Phase 间承接与镜头过渡方案

以 15s 为一 phase 构建剧本时，除「上一段尾帧 → 下一段首帧」V2V 外，**在创建剧本阶段**就为每对相邻 phase 标注可选过渡方案。

### 可选方案

| 方案 | 说明 | 在剧本中的落点 |
|------|------|----------------|
| **首尾帧衔接** | 上一段最后一帧 = 下一段视觉起点，V2V 连续 | 00/本段素材：尾帧图；提示词：same scene, consistent character |
| **动作切点** | 在动作自然断点处分段 | 情节说明/提示词：段尾以「手放杯结束」、段头以「转身后开始」等 |
| **景别/镜头变化** | 段界处换景别（中景→特写、空镜再回人） | README/00：标出段 N 景别；提示词写清景别 |
| **黑场/淡入淡出** | 段间插入短黑场或淡出淡入 | README 剪辑时间轴或 phase 备注 |
| **情绪/节奏切点** | 用节奏或情绪转折作软过渡 | 00/README：标出段 5→6 节奏由紧到松等 |
| **声音/台词桥梁** | 用台词或环境声跨段 | 段尾收束句，段头起句；或标注可接 BGM 过渡 |

### 使用要点

1. 在 00 或 README 中做 **phase 衔接总表**：列出每对相邻 phase 采用的过渡方式。
2. 在 phase 内写清边界：段尾保留动作/静帧/景别；下一段注明段头从何动作/景别开始。
3. 若当前仅支持首尾帧 V2V，其他方案可在剪辑阶段实现，但剧本里先标好。

## 2. 当前 phase 内提及前后段（环环相扣）

**当前 phase** 的剧本与提示词应**部分提及**上一 phase 尾部与下一 phase 首部，使段与段环环相扣，避免观感断裂或状态跳变。

### 改进要点

1. **【本段情节说明】**：段头用一两句点明「承接上段末尾的什么状态/动作」；段尾点明「本段收束在什么状态，为下段什么开头做铺垫」。
2. **Seedance 提示词**：段头可带一句「承接…」；段尾可带一句「过渡到下一段…」；若平台字数敏感，至少情节说明的「前后提及」必做。
3. **与镜头过渡配合**：前后提及与衔接总表中的动作切点、景别、情绪切点一致。
4. **台词跨段不重复**：写本段对话时考虑上一 phase 已说了什么，避免同一句或同一意图在相邻段重复；写前扫一眼上段【本段剧本台词】。

### 自检清单（环环相扣）

- [ ] 本段情节说明是否包含「承接上段…」（段 2～N）与「本段以…结束」「下段将…」（段 1～N-1）。
- [ ] 提示词（若允许）是否在段头或段尾有一句前后衔接说明。
- [ ] 本段台词是否与上段已出现的句子/意图重复；若有是否已改成不同说法或删减。

## 3. V2V 切镜边界与 phase 时长

### Section 内 V2V 切镜边界（每最多 4 phase 一断）

**问题**：连续多段都用尾帧→首帧 V2V，画面噪声逐段扩散、画质变差。

**规则**：Section 内**最多每 4 个 phase 需一次「切镜」**——保持环境/场景不变，但**不再使用 V2V**，下一段用新镜头/新起始画面。

**剧本设计**：在交界处（如 phase04 尾、phase08 尾）安排段尾**画面较为平稳**；下一段（phase05、phase09 等）在 00/README 或提示词中标注为「切镜段：不用上段尾帧 V2V」。

可与「景别/镜头变化」「动作切点」一起规划：在 phase 4、8、12… 采用「非 V2V」过渡。

### Phase 时长不固定为 15 秒

**问题**：默认每段 15s 易为凑时长稀释内容或为压内容挤满 15s，节奏与剧本脱节。平台支持 **4～15s**。

**规则**：**按剧本对话密度/内容密度设定每 phase 时长**，不必每段都 15s。例如：对话密集段可 6～8s；留白/氛围段可 12～15s。

**剧本设计**：在 00 或 README 为每个 phase 标注**建议时长**（4～15s 内或「短/中/长」），情节与台词时间块按该时长分配。

### 自检清单

- [ ] 本 section 是否在 phase 4、8、12… 做了切镜规划？交界处上一段尾是否设计为画面较平稳？
- [ ] 各 phase 是否在 4～15s 范围内按对话/内容密度设定时长，00/README 是否标建议时长或短/中/长？
