# 四、动作 / 神态 / 细节描写

### 问题简述

- 情节说明与提示词多用概括句（如「青年抱臂表情严肃，少女憋笑偷看他」），缺少**可被画面执行的**动作链与神态变化。
- 结果：模型难以生成细腻的肢体与表情，人物活动性低、互动显得呆板。

### 改进要点

1. **情节说明**：不只写「谁做了什么」，补上**动作顺序、视线、微表情**。  
   例：不说「蕾娜憋笑偷看」，而写「蕾娜先抿嘴憋笑、视线瞟向他，再忍不住嘴角上扬、用手背挡一下嘴」。
2. **Seedance 提示词（本段部分）**：  
   - 把「一句话概括」拆成**多个可执行动作**：谁先动、谁后动、手/身体/视线如何配合。  
   - 加入**表情变化**：从什么表情到什么表情（如从微蹙眉到憋笑）。  
   - 两人互动时写清**触发与反应**：A 做 X → B 做 Y（如青年抱臂 → 少女瞥他一眼再抿唇憋笑）。  
   - **粘贴到 Seedance 的块**须按《phase-prompt-format》采用**五段式结构**；其中【本段动作与节奏】用**时间块**（开场/中段/收尾）写主要动作与节奏，**不要**把整张秒级表粘贴进提示词框——秒级表保留在 phase 文件内供分镜与配音参考，模型更易从结构化时间块中学习。
   - **秒级只用线性表述、不用表格**：秒级动作只保留**一份**，用「0-1s 青年…；少女…。1-2s …」的句式写在【本段动作·演出·运镜·场景（秒级）】内，不另做表格、不重复第二份。可**按时间块换行**（一行一块）便于人读；整份给 AI 或粘贴时若换行被清除，仍可凭「0-1s」「1-2s」等解析。
3. **与平台限制平衡**：若整份粘贴且平台对英文敏感，提示词用中文时同样要写细动作与神态，避免只写「抱臂」「憋笑」等笼统词。

### 自检清单（动作/神态）

- [ ] 本段情节说明里是否有「动作顺序 + 视线/微表情」而不只有一句概括。
- [ ] 本段提示词里是否有「A 做 X、B 做 Y」的互动链，而不是仅「A 和 B 在做什么」。
- [ ] 是否包含至少一处「表情变化」或「从…到…」的神态描述。

## 寡言/内敛人设与可执行微反应（section03 中期复盘）

**问题简述**

- **人设**（如沉稳、寡言、内敛）描述的是**性格与台词量**，不等于**可被画面执行的反馈**。若剧本与提示词只强调「少言」「不张扬」而未显式写出该角色对另一方的**微反应**（视线、眼珠微动、喉结/眉/嘴角的极微变化、呼吸/肩/手指等肢体小动作），生成模型易将「寡言」解读为「少动/无表情」，成片呈现为双人戏中一方活跃、另一方近乎静态。
- 表现：观众反馈「男主没有任何微表情，神态/情绪/肢体都没有和女主产生互动」。

**改进要点**

1. **人设与可执行动作的区分**  
   - 在 00/README 的**角色概述**中明确：寡言/内敛 = 少台词、不夸张、不抢戏，**不意味着**无表情、无反应、无互动。  
   - 约定：该角色须有对对手方的**持续可执行微反应**（听对方说话时的视线/眼珠微动、喉结微动、呼吸/肩动、手指或道具上的小动作、被逗到时眉或嘴角的极轻微变化、不躲开对方靠近等），且**对手方有动作/台词时，该角色必须有可被画面执行的回应**（可克制，但需写出）。

2. **秒级表与触发-反应链**  
   - 秒级表中，寡言/内敛角色所在列（如「青年」）**每一时间块**都需写出**对对手方动作/台词的回应**，避免仅「保持」「同左」「静止」等被动描述。  
   - 采用**触发-反应**表述：对手做 X 的时段内，该角色的 Y 反应（例：她说「哼—」→ 他眼珠向她方向微动约 5 度、喉结微动；她靠过来 → 他手指在栏杆上微动、不躲）。  
   - 微反应类型须具体可执行：眉/眼/嘴角/喉结、视线方向与幅度、手指/肩/呼吸、下颌线放松等。

3. **提示词【本段动作与节奏】中的双向互动**  
   - 粘贴到 Seedance 的【本段动作与节奏】须**交替或成对写出**「A 做 X → B 做 Y」，不能只强调一方（如仅写少女的台词与动作）。  
   - 显式写出寡言/内敛角色的反应节奏（例：「少女说哼—侧头看他，**青年眼珠微向她动、喉结微动仍不转头**；少女看他侧脸笑意明显，**青年望夜景呼吸一次、肩微动**」），使模型明确该角色「有在听、在看、在反应」，仅以克制的微表情与肢体表达。

**自检清单（寡言/内敛角色）**

- [ ] 00 或 README 角色概述是否澄清：寡言/内敛 ≠ 无表情无互动，需有对对手方的可执行微反应。
- [ ] 本段秒级表该角色列是否每一时间块都有**对对手方动作/台词的具体回应**，而非仅「保持」「同左」「静止」。
- [ ] 【本段动作与节奏】是否显式写出「对手做 X → 该角色做 Y」的节奏，且 Y 为可被画面执行的微反应。

## 台词/对白在提示词中的强化（避免 Seedance 丢失配音）

**问题简述**

- 当前 phase 的【本段剧本台词（配音/TTS 用）】主要供配音/TTS 使用，粘贴到 Seedance 时模型易将其视为「参考」而非「须体现的对白」，导致成片口型/台词缺失或错位。
- 【台词概述】多为一句概括（如「少女说辛恩你总是这样呢、什么都不说就做了」），缺少时间轴与逐句台词，模型难以绑定「谁在何时说什么」。

**改进要点（提示词优化）**

1. **在粘贴块内增加「本段对白（须体现）」**  
   - 在五段式结构中，将【台词概述】升级或紧接增加**【本段对白（须体现口型）】**：按时间块列出「谁在何时说什么」，用中文写清每句台词，便于模型生成口型与对白。  
   - 格式示例：「0-5s 少女说：辛恩你总是这样呢。6-10s 少女说：什么都不说就做了。11-14s 少女说：……你要是说一声我也……（欲言又止）。」  
   - 段首可加一句锚定：「本段须体现以下对白与口型，人物说话时口型与台词一致。」

2. **【本段动作与节奏】中嵌入口型/台词**  
   - 在动作描述里**直接写出具体台词**，而不是只写「少女说了一句调侃」：如「少女头转向青年说「辛恩你总是这样呢」，再说「什么都不说就做了」笑意明显」。  
   - 让「谁在说哪句话」与动作同句出现，提高模型把台词绑定到口型的概率。

3. **负面词补充（可选）**  
   - 若平台支持且不违禁，可加「不要静音、不要无口型、不要嘴不动、不要只有表情不说话」，降低「只动表情不开口」的生成。

4. **无台词段落**  
   - 若本段无对白（仅动作/呼吸/环境），在【本段对白】或【台词概述】中明确写「本段无台词，仅动作与环境音」，避免模型误补无关对话。

**自检清单（台词/对白）**

- [ ] 粘贴到 Seedance 的块内是否有**按时间块的本段对白**（谁、何时、说什么），而不只有一句台词概述。
- [ ] 【本段动作与节奏】中是否在描述动作时**写出关键句台词**（引号内），而不只写「她说了一句」「他应声」。
