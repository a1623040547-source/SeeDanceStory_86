# 三、场景环境与空间（环境设定与人景位置）

本主题合并：**环境氛围打光**、**环境与门的设计**、**人与场景位置关系**。用于自检「场景长什么样、人在哪」。

---

## 1. 环境/氛围/打光统筹

### 问题简述

- 整体环境、打光、窗外天气随 phase 漂移（如第一段暖光+雨窗，后面变冷光/下雪、画面变花），与段 1 确立的氛围不一致。

### 改进要点

1. **在提示词中统筹全片环境**：在总述/规则与每段提示词中明确本 section 的固定环境（如 warm indoor lighting, rainy window, same cozy atmosphere throughout）；每段可重复一句环境锚定（Same warm lighting and rainy window as segment 1; no snow, no cold blue tint）。
2. **提供「第一段环境氛围参考图」**：从段 1 生成视频中选取代表氛围的一帧，放入 refs（如 refs/env_atmosphere/ 或 refs/scenes/）；段 2～12 生成时除尾帧外，若平台支持多图，**额外上传该环境参考图**以保持光线、窗外天气与色调一致。
3. **在 00/素材说明中写清**：环境氛围参考图的用途与使用段（段 2～12）。

### 自检清单（环境/氛围）

- [ ] 00 或 README 是否写明本 section 的**统一环境设定**（暖光、雨窗、无雪、室内等）。
- [ ] 是否已从段 1 提取并保存**环境氛围参考图**至 refs；段 2～12 提示词是否包含与环境一致的锚定句；若支持多图，是否注明「上传环境氛围参考图」。

## 2. 环境与门的设计（Seedance 适配 + 场景细致度）

### 门：一律用自动门（横向滑动进出）

- **原因**：Seedance 难以正确执行「用手开门」（推/拉/转门把）；成片易手与门关系错位或门无动于衷。
- **约定**：凡涉及「门」的进出，一律改为**自动门（横向滑动进出）**。写法示例：「自动门」「横向滑开/滑闭」「人物走向门、门横向打开、人物走出」。避免：推门、拉门、转动门把、扶门。
- **在 phase 提示词中**：写「两人走向玄关、自动门横向滑开、走出」等，不写「推开门」「拉开门」。

### 环境描写必须细致（像小说家一样交代场景）

- **问题**：若环境仅一句笼统概括，各 phase 画图/生成时细节不一致，人景关系难延续。
- **改进**：环境设计要达到**可据此得到同一套空间与氛围**的程度。包括：**空间布局**（门、窗、家具方位）；**具体物件**（门/窗/矮柜/沙发/桌的位置、颜色、形状）；**材质与色调**（地板、墙面、冷暖明暗）；**光线**（主光源方向、明暗差异）；**动线与锚点**（从门到窗的路径、可站可坐可放行李的位置）。
- **落实位置**：各 section 的 refs/scenes/README 或 00 中应有**完整、细致场景描述**；phase 提示词【场景与氛围】引用或改写该描述，保证段与段场景细节一致。

### 自检清单（门与环境）

- [ ] 本 section 若有「门」的进出，是否已改为**自动门（横向滑开/滑闭）**，且无推门/拉门/转门把。
- [ ] refs/scenes/README 或 00 是否有**完整、细致的场景描述**（空间布局、门/窗/家具、材质色调、光线、动线）；各 phase【场景与氛围】是否与之一致。

## 3. 人与场景位置关系

### 问题简述

- Seedance 对空间位置理解有限；有运镜时人物与场景的**相对位置**易错位或漂移（如窗左/沙发前关系混乱）。

### 改进要点

1. **在 phase 提示词中明确「人与场景的位置关系」**：不只写「两人在客厅」，要写清人物在空间中的锚点（如「男主站在窗左侧、面向室内；女主坐在沙发右侧，与男主相对」「茶几在两人之间；镜头从男主肩后看向女主」）。与景别、运镜一起写时，位置关系要一致。
2. **与「初始化环境」配合**：若本 section 有第一段环境氛围参考图或环境初始化描述，其中应包含**空间布局**约定；各 phase 提示词中的位置描述与该布局一致。
3. **运镜时显式约束**：有推/拉/摇/移时，补一句运镜过程中保持不变的关系（如「横摇时保持男主在画左、女主在画右，两人相对位置不变」）。
4. **通过尾帧观察空间关系**：写下一段或上传参考前，**查看上一段尾帧**，确认人、物在画面中的实际位置；下一段提示词与尾帧一致，若尾帧与剧本有偏差，以尾帧为准调整提示词。

### 有运动/动作处须写清「怎样运动」（位置与动作具体化）

- **问题**：仅用概括词（如「泼水」「回击」「撩水」「挡」「躲」）而不写清身体怎样动，Seedance 易生成不符合物理或观感别扭的画面。典型：玩水时**女主**若写「弯腰/蹲低用手撩水或捧水泼向」则易生成合理动作，**男主回击**若只写「反手撩水」「回击」而无具体动作描述，成片常缺明确肢体动作。
- **改进**：凡有**明显运动**的段落（泼水、撩水、挡水、躲、扶、拨头发等），在情节说明与提示词中写清：
  - **谁**、**身体哪部分**、**怎样动**（如弯腰、蹲低、手臂从身侧/腰高撩水向对方、手掌拨水、抬手挡脸/挡身前、侧身躲、伸手扶肩/扶腰）。
  - 避免仅用「泼水」「回击」「挡」「躲」一笔带过；要可执行、可想象出具体画面。
- **示例**：少女**弯腰或蹲低**，**用手撩水或捧水**泼向少年；少年**抬手挡在身前或挡脸**，或**侧身躲开**。少年回击时**手臂从身侧或腰高撩水向少女**、**手掌拨水**（轻量、玩笑感），而非只写「反手撩水」。

### 自检清单（位置关系）

- [ ] 每段提示词是否包含**人物在场景中的具体位置**（窗左/沙发右/两人相对），而非仅「在客厅」「在室内」。
- [ ] 有运镜的 phase 是否显式约束运镜过程中人景/人人相对位置不变（或按剧本写清合理变化）。
- [ ] 环境/空间初始化是否约定**空间布局**，各 phase 是否与之一致。
- [ ] 写下一段或上传参考前是否**查看上一段尾帧**，根据实际人景位置撰写/调整本段提示词。
- [ ] **有运动/动作的段落**（泼水、撩水、挡、躲、扶等）是否已写清**怎样运动**（弯腰/蹲低、手捧水/撩水、抬手挡、侧身躲、手臂从哪撩向哪等），而非仅用「泼水」「回击」等概括词。
