# 十八、禁止模糊表述（明确指向）

**规则**：在情节说明、位置关系、物品关系、提示词等**任何涉及剧情设计的位置**，都不要使用「A 或 B」「A/ B」等模糊描述，必须**明确指向一种**（是 A 就是 A，是 B 就是 B）。否则视频生成会在不同镜头或不同 phase 中做出不一致的选择，导致画面错乱或物品/动作前后矛盾。

### 反例（section01 phase05/06 复盘）

- 「蕾娜拿**帽子或小包**」「手持**帽或包**」「**帽子或包**」：生成时可能这段拿帽、下段拿包，或同一段内前后不一致。
- 改进：台词已说「帽子、持ってくね」→ 全 section 统一为**帽子**；情节说明、物品、提示词一律写「帽子」「持帽子」，不写「帽或包」。

### 正确做法

- 物品：确定是帽子就写「帽子」，是小包就写「小包」；不在同一句里写「帽子或包」。
- 动作/位置：确定一种写法（如「向放帽子处走」），不写「向放帽/包处走」或「持帽或包」。
- 若创作阶段尚未定稿，先**在 00 或 README 中定稿**（如「本 section 蕾娜手持物为帽子」），再在各 phase 中统一使用该表述。
