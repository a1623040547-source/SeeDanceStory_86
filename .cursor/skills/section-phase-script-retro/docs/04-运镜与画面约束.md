# 四、运镜与画面约束（镜头与画面技术层）

本主题合并：**运镜镜头程度**、**画面人数明确**、**走路与位移**。用于自检「景别/运镜怎么写、画面几人、走/站怎么区分」。

---

## 1. 运镜 / 镜头 / 程度说明（即梦向）

### 问题简述

- 提示词缺少具体运镜效果、镜头说明、程度说明，仅用「镜头稳定」「治愈氛围」等笼统表述，不利于平台精准控效。即梦推荐：**主体动态 + 运镜类型（速度/角度）+ 场景变化 + 风格**。

### 改进要点

1. **每段提示词明确「景别」**：用标准景别词——特写（面部/手部/物品）、近景（胸部以上）、中景（膝上/半身）、全景（全身）、远景（环境/空间）。例：「中景，双人半身，沙发与茶几在画内」。
2. **明确「运镜类型」与「程度」**：固定机位写「固定机位」「无运镜」「static camera」；推/拉/摇/移写清类型与程度（缓慢推进、轻微推进、从全景推到中景等）；程度词用缓慢、轻微、微微、逐渐、匀速。
3. **与官方示例对齐**：介绍细节用特写，展现环境用全景，欢乐/互动可用跟拍或轻微运镜；若有明确运镜可写「首帧：…；运镜：…；尾帧：…」。
4. **在 00 或 README 做「本 section 运镜约定」**：如「室内日常以固定机位、中景为主」「段 N 为特写」等，各 phase 与之一致。

### 自检清单（运镜/镜头）

- [ ] 每段提示词是否包含**景别**（特写/近景/中景/全景/远景至少其一）及**运镜类型+程度**。
- [ ] 00 或 README 是否有一句「本 section 运镜约定」或景别/运镜总表。

## 2. 画面人数明确（避免同一人重复出现）

**规则**：每个 phase 的提示词中必须**明确写出本段画面里有几个人**（如「画面内仅两人」「only two people in frame」），避免 Seedance 重复生成同一角色导致**同一人在同一画面中出现两次**。

### 正确做法

- 在【项目·风格·规则】或【位置关系（本段）】中显式写一句人数，例如：「画面内仅两人：少女与少年；无其他人物。」双人戏建议每段都带「两人」「仅二人」「two people only」等。

### 自检清单

- [ ] 本 phase 的 Seedance 提示词是否已写明**画面内人数**（如「仅两人」），且与角色设定一致（双人戏即两人、单人镜头即一人）？

## 3. 走路与位移（避免原地踏步感）

**规则**：凡涉及「走」「踩水」「玩水时移动」等，须**明确是「有位移的前进」还是「站定/位置基本不变」**，否则易生成**原地踏步**（迈步但相对背景无位移）。

### 改进要点

1. **站定、不前进**（如浅滩挽臂踩水、站在水中玩水）：在提示词中显式写**位置基本不变、不向前行走**；**避免出现原地踏步或原地走路的动作感**；可补充「站在水中、脚可轻动踩水但两人整体位置不变」。
2. **有位移的走路**（如沿沙滩走、沿浅滩向前走）：显式写**沿某方向向前走**；**人物相对背景有明显位移/背景相对后移**；**不是原地踏步**；可补充「双人并肩或挽臂向前行进、画面中可见身体与沙滩/海水的相对移动」。
3. **与景别/运镜配合**：固定机位 + 有位移时人物应从画内一侧向另一侧或向纵深移动；固定机位 + 站定踩水时人物与背景相对位置不变，仅脚部/身体轻微动作。

### 自检清单

- [ ] 本段若有「走」「踩水」「玩水时移动」等，是否已区分「站定不前进」与「向前行走」？
- [ ] 站定是否已写「位置基本不变、不向前行走」「避免原地踏步感」？有位移是否已写「沿某方向前进」「相对背景有位移」「不是原地踏步」？

## 4. 动作与运动须具体化（有运动处写清怎样动）

**规则**：凡涉及**明显肢体运动**（泼水、撩水、挡水、躲、扶、拨头发等），提示词须写清**身体怎样动**，不能只用「泼水」「回击」「挡」「躲」等概括词，否则 Seedance 易生成不符合物理或观感差的画面。

- **典型问题**：玩水段中女主「弯腰撩水」有具体描述则成片自然；男主「回击」若只写「反手撩水」而无手臂/手部动作描述，成片常缺清晰动作。
- **改进**：在【本段动作与节奏】与【位置关系】中写清：谁、身体哪部分、怎样动（弯腰/蹲低、手捧水/撩水向对方、抬手挡脸或挡身前、侧身躲、手臂从身侧或腰高撩水等）。详见 [03-场景环境与空间.md](03-场景环境与空间.md)「有运动/动作处须写清怎样运动」。
