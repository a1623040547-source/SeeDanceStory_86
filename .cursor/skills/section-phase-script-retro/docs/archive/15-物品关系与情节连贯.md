# 十五、物品关系与情节连贯（跨 phase 状态一致）

### 问题简述（section01 复盘）

- **跨 phase 物品状态不一致**：某件「物品」或「动作」在前一段已经完成，后一段剧本又重复描写，导致情节逻辑断裂、观感重复。
- **本例**：story/01-海边度假/section01 中，**phase02** 已明确「辛恩把行李放在门边或矮柜」「放好行李后视线朝她」「本段以蕾娜望海、辛恩放好行李结束」；**phase04** 却再次写「辛恩点头说『好』并把最后一件行李放好」「收尾：少年把最后一件行李放好」。行李在 phase02 已放好，phase04 不应再出现「放行李」动作。
- **根因**：写单段时未沿时间线检查**前面各 phase 中该物品/动作是否已发生**，只按「本段想写的情绪与台词」写了动作，未做跨段物品与情节的核对。

### 创作时的三类关系

剧本创作（含情节说明与 Seedance 提示词）时，需同时考虑：

1. **位置关系**：人物在场景中的空间位置、与景/物的相对关系；见 [14-人与场景位置关系.md](14-人与场景位置关系.md)。
2. **情节关系**：前后段情节先后、情绪与节奏衔接；见 [06-环环相扣.md](06-环环相扣.md)。
3. **物品关系**：**物品在剧情中的状态与归属**（如行李是否已放下、帽子是否已拿、杯子是否已放下）、**动作是否已在更早的 phase 完成**，避免同一动作或同一物品状态在多段中重复或矛盾。

### 改进要点

1. **建立「物品/动作时间线」**
   - 在 section 的 00 或 README 中，对关键物品/动作做简要标注：例如「行李：phase02 门边放好；帽子/包：phase04 末拿、phase05 出门」。
   - 写每一段时先扫一眼：本段要写的动作或物品，是否已在更早的 phase 完成或改变状态；若已完成，本段不再写「再次完成」或改为写其它收束动作。

2. **情节说明与提示词中显式承接「物品状态」**
   - 若前段已有「放好行李」「放下杯子」「拿起包」等，本段情节说明应写「承接上段行李已放好/包已拿起…」，本段动作只写**之后**发生的事（如点头、转身、出门），不再写「放好行李」等已完成的动作。

3. **写新 phase 前的自检**
   - 写 phase N 时，快速过一遍 phase 1～N-1 的【本段情节说明】与【本段动作与节奏】，列出「已出现的物品与已完成的动作」清单，避免本段重复或矛盾。

### 自检清单（物品关系与情节连贯）

- [ ] 本段是否写了某件「放/拿/用」类动作，而该动作或该物品状态已在**更早的 phase** 完成或改变？若有，删除或改写本段中的重复动作。
- [ ] section 内是否有**关键物品/动作**（行李、包、帽子、外套、杯子等）的简单时间线或标注（在哪一段放下/拿起/穿脱），供写后续段时对照？
- [ ] 本段情节说明是否与「承接上段…」一致：若上段已写「放好行李」，本段不应再出现「把行李放好」，而应写「行李已在门边，本段…」。
