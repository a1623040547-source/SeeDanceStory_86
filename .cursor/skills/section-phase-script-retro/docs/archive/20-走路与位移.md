# 二十、走路与位移（避免原地踏步感）

**规则**：提示词中凡涉及「走」「踩水」「玩水时移动」等，须**明确是「有位移的前进」还是「站定/位置基本不变」**，否则 Seedance 易生成**原地踏步、原地走动**（人物在画内迈步但相对背景无位移），观感错误。

### 问题来源（section02 中期复盘）

- 海边玩水、浅滩踩水等段落中，若只写「踩水」「挽臂走」等，成片易出现两人**在原地走动**（有迈步动作但无前进位移）。
- 模型易把「踩水/玩水」理解为「走路」，又未约束「往哪走」，结果变成原地踏步。

### 改进要点

1. **若本段是「站定、不前进」**（如浅滩挽臂踩水、站在水中玩水）  
   - 在提示词中显式写：**位置基本不变、不向前行走**；**避免出现原地踏步或原地走路的动作感**。  
   - 可补充：站在水中、脚可轻动踩水但两人整体位置不变；若有迈步感须为站定处的轻微踩水、非沿沙滩/水面位移。

2. **若本段是「有位移的走路」**（如沿沙滩走、沿浅滩向前走）  
   - 在提示词中显式写：**沿某方向向前走**（如沿沙滩、沿水边）；**人物相对背景有明显位移/背景相对后移**；**不是原地踏步**。  
   - 可补充：双人并肩或挽臂向前行进、画面中可见身体与沙滩/海水的相对移动。

3. **与景别/运镜配合**  
   - 固定机位 + 有位移的走路：人物应从画内一侧向另一侧（或向纵深）移动，背景相对变化。  
   - 固定机位 + 站定踩水：人物与背景相对位置不变，仅脚部/身体轻微动作。

### 自检清单

- [ ] 本段若有「走」「踩水」「玩水时移动」等，是否已区分「站定不前进」还是「向前行走」？
- [ ] 站定不前进时，是否已写「位置基本不变、不向前行走」「避免原地踏步感」？
- [ ] 有位移走路时，是否已写「沿某方向前进」「相对背景有位移」「不是原地踏步」？
