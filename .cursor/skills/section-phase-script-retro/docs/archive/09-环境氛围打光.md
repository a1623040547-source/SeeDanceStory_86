# 九、环境/氛围/打光统筹

### 问题简述（section02 后期复盘）

- 虽保证了**上一 phase 尾帧 → 下一 phase 首帧**的 V2V 衔接，但**整体环境、打光、窗外天气**随 phase 往后发生漂移。
- 表现：第一段时环境光较暖、窗外为下雨，越到后面越易变为下雪或冷光，画面越来越花，与段 1 确立的氛围不一致。

### 改进要点

1. **在提示词中统筹全片环境/氛围**
   - 在**总述/规则**与**每段提示词**中明确写出本 section 的固定环境设定，例如：`warm indoor lighting, rainy window (not snow), same cozy living room atmosphere throughout`。
   - 每段本段描述里可重复一句环境锚定：如「Same warm lighting and rainy window as segment 1; no snow, no cold blue tint.」

2. **提供「第一段环境氛围参考图」到 refs**
   - **来源**：从段 1 生成视频中选取一帧能代表「暖光、雨窗、室内氛围」的画面（或直接用段 1 首图/尾帧），保存为环境氛围参考图。
   - **放置**：放入 `refs/env_atmosphere/`（或 `refs/scenes/` 并单独说明），命名如 `section02_env_ref.png`。
   - **使用**：段 2～12 生成时，除上传「上一段尾帧」外，若平台支持多图，**额外上传该环境氛围参考图**作为「环境/场景/打光」参考，使模型在角色与动作接力的同时，保持与段 1 一致的光线、窗外天气与整体色调。

3. **在 00 / 素材说明中写清**
   - 在「素材类型与衔接规则」中增加**环境氛围参考图**一类，说明其用途与使用段（段 2～12）。
   - 在「每段使用的素材一览」中注明：段 2～12 除尾帧外，建议（或必须）使用环境氛围参考图。
