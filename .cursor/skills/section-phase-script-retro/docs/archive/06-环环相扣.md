# 六、当前 phase 内提及前后段（环环相扣）

为保证剧情与人物形象衔接自然，**当前 phase** 的剧本与提示词应**部分提及**上一 phase 的尾部故事与下一 phase 的首部故事，使段与段之间环环相扣，避免「每段只写本段」导致的观感断裂或人物状态跳变。

### 改进要点

1. **【本段情节说明】**  
   - 段头：用一两句点明「承接上段末尾的什么状态/动作」（如「承接上段蕾娜整盘端走后」）。  
   - 段尾：用一两句点明「本段收束在什么状态，为下段什么开头做铺垫」（如「以辛恩板脸瞥她结束，下段蕾娜憋笑破功」）。  
   - 不必长，目的是让读者与生成方明确「从哪接到哪」。

2. **Seedance 提示词（本段部分）**  
   - 开头可带一句「承接」：如「Continuing from previous segment: young woman has just pulled the plate to her side. This segment: ...」或中文「承接上段少女将盘子挪向自己后。本段：…」。  
   - 结尾可带一句「过渡到下一段」：如「Segment ends with young man's stern look; next segment will open with young woman breaking into laughter.」或「本段以青年板脸结束，下段由少女笑场开场。」  
   - 若平台字数敏感，可仅在情节说明中写清，提示词保持简洁，但情节说明的「前后提及」必做。

3. **与「镜头过渡」配合**：上一段尾部/下一段首部的提及，与 phase 衔接总表中的动作切点、景别、情绪切点一致，避免说法矛盾。

4. **台词/对话跨段不重复**（section01 海边度假复盘）  
   - 构建**当前 phase 的对话**时，须考虑**上一 phase 已经说了什么**，避免同一句或同一意图在相邻段重复，造成观感奇怪（如段 6 辛恩已说「嗯、走吧」，段 7 辛恩不再重复「走吧」，可改为「出去吧」或只认同天气）。  
   - 写本段台词前快速扫一眼上段【本段剧本台词】，确保接话、认同、行动类短句不与上段撞车。

### 自检清单（环环相扣）

- [ ] 本段情节说明是否包含「承接上段…」或「上段末尾…」的表述（段 2～N）。
- [ ] 本段情节说明是否包含「本段以…结束」「下段将…」或「为下段…做铺垫」的表述（段 1～N-1）。
- [ ] 提示词（若允许）是否在段头或段尾有一句前后衔接说明。
- [ ] 本段台词是否与上段已出现的句子/意图重复（如「走吧」连两段由同一人说或接同一句）；若有，是否已改成不同说法或删减。
