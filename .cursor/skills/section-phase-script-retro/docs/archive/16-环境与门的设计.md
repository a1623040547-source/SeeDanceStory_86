# 十六、环境与门的设计（Seedance 适配 + 场景细致度）

### 一、门：一律用自动门（横向滑动进出）— 中期复盘

- **原因**：Seedance 生成视频时，若未给出非常明确、分步的提示，**人物往往无法正确执行「用手开门」**（推门、拉门、转门把、扶门框等）。成片易出现手部与门的关系错位、或门无动于衷、人物动作悬空等。
- **约定**：后续设计环境时，**凡涉及「门」的进出，在描述与提示词中一律改为「自动门（横向滑动进出）」**。
  - 写法示例：**「自动门」「横向滑开/滑闭」「人物走向门、门横向打开、人物走出」**。
  - 避免写法：推门、拉门、转动门把、扶门、关门（若为自动门则写「门横向滑闭」）。
- **在 phase 提示词中的落实**：写「两人走向玄关、自动门横向滑开、走出」或「少女转身走向自动门、门横向滑开、她进入室内」，**不写**「推开门」「拉开门」「她开门进屋」。

### 二、环境描写必须细致（像小说家一样交代场景）

- **问题**：若环境仅用一句笼统概括（如「民宿室内、玄关或带窗房间、木质或白墙、简洁家具」），各 phase 或不同 section 在画图/生成时容易在细节上不一致，人物与场景的位置、道具与光线也难保持连贯。
- **改进**：**环境设计要达到「像小说家一样把故事发生的场景细节交代清楚」**，使任何人（或图生/视频生成）都能据此得到同一套空间与氛围。
  - **空间布局**：玄关与房间的关系（开放一体 / 短通道）；门、窗、家具分别在什么方位（如门在画右、窗在正面、矮柜在门侧）。
  - **具体物件**：门——自动门、横向滑动、门框材质与颜色；窗——大小、是否落地、窗外可见什么（海平线、天空、有无阳台、树梢）；矮柜/鞋柜/衣架/沙发/桌——位置、颜色、大致形状与数量。
  - **材质与色调**：地板（木纹/瓷砖/颜色）、墙面（白/米白/浅灰）、天花板感；整体冷暖与明暗。
  - **光线**：主光源方向（如从窗照入）、在玄关与窗边的明暗差异、是否有次要光源（如壁灯）。
  - **动线与锚点**：人物从门到窗的路径；可站、可坐、可放行李的具体位置（如「行李放在门右侧矮柜旁」）。
- **落实位置**：
  - 各 section 的 **refs/scenes/README.md**（或等价场景说明）应包含**该 section 的完整、细致场景描述**，而非仅引用 story 00 的一句「环境描述」。
  - Story 00 中「各 Section 环境描述（用于生成环境图）」可保留简明版，但**详细版应以各 section 的 refs/scenes 或 00 中的「场景细致描述」为准**。
  - 写 phase 提示词时，【场景与氛围】可直接引用或改写该细致描述，保证段与段之间场景细节一致。
