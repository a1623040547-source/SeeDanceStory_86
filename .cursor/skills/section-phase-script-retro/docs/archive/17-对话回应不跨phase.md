# 十七、对上一句的回应不放到下一 phase（Seedance 对话关系）

**规则**：不要把**对上一句台词的回应**放到下一个 phase。否则 Seedance 无法在同一段内建立「说—应」关系，导致对话关系与口型/节奏处理不当。

### 原因

- 每个 phase 是独立 15s 生成单元；下一 phase 以**上一段尾帧**做 V2V 延续的是**画面**，不是「上一句台词」的上下文。
- 若 A 在 phase N 末说「终于啊」，B 在 phase N+1 开头说「嗯」接住，模型在 N+1 里看不到 A 的台词，容易把 B 的「嗯」处理成无明确对象的反应或口型/节奏错位。

### 正确做法

- **同一组「陈述/问句 + 回应」尽量放在同一 phase 内**（如辛恩「终于啊」与蕾娜「嗯」放在同一段）。
- 若因节奏或时长必须切段，则**把回应收在上一个 phase 的段尾**（如 phase06 尾 0～3s 留出蕾娜「嗯」再接段尾动作），下一 phase 从新话题或新动作开始。

### 反例（section01 phase06 / phase07 复盘）

- **phase06** 段尾：辛恩 9～12s「终于啊」；12～15s 无台词、两人出门。
- **phase07** 段头：蕾娜 0～4s「嗯」接住「终于啊」、带终于等到这一天的感觉；随后辛恩提议泳装。

问题：蕾娜对「终于啊」的回应在下一段，Seedance 在生成 phase07 时无法正确关联「嗯」与上一段的「终于啊」，影响对话关系与口型。  
改进：将蕾娜「嗯」前移到 phase06 段尾（如 12～14s），或把「终于啊」+「嗯」都放在 phase06 内再结束；phase07 从辛恩看穿着、提议泳装开始。

- [ ] 写新 phase 时是否扫过上段【本段剧本台词】最后一句，避免本段 0～4s 出现「接上句」的对话？
