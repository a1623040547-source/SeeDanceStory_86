# 十二、穿着/服装状态跨 phase 一致性（section03 中期复盘）

### 问题简述

- 剧情中有**穿着变化**（如男主脱外套给女主）时，若剧本与提示词只写「本段动作」而不写**当前穿着状态**，下一段或后续生成中模型可能让角色恢复为「默认」造型，导致与剧情矛盾（例如男主又穿上外套）。
- **另一可能原因**：上传的**人物形象参考图**中若角色穿着外衣，Seedance 在合成时可能把 ref 里的外衣强加回角色身上，覆盖剧情中的「已脱外套」状态。

### 改进要点

1. **在相关 phase 中显式写清穿着状态**  
   - 从「脱外套」发生的那一段（如 phase04 尾）之后，直至外套归还或场景结束，**每一段的【本段情节说明】或提示词**中应包含一句**穿着锚定**：  
     - 例：「本段内：男主已脱外套，只着内搭/衬衫；女主披着男主外套。」  
   - 可放在【本段动作与节奏】的开场句或情节说明首句，便于模型在每段开头就锁定造型。

2. **人物形象参考图与剧情阶段匹配**  
   - 若当前使用的**人物形象参考图**中男主穿着外套，而剧情已进入「外套在女主身上」的阶段，有两种做法：  
     - **做法 A**：为「脱外套后」的段落单独准备一张**无外套**的男主 ref（同一角色、同画风，仅去掉外套），在该段及后续生成时使用。  
     - **做法 B**：不换 ref 时，在提示词中**反复强调**本段造型（如「男主未穿外套、仅内搭」「男主无外衣」），弱化 ref 中外套对生成结果的影响。

3. **自检清单**  
   - 凡 section 内有**明显穿着变化**（脱衣、披衣、换装等），在 00 或 README 中标出「从 phase N 起：男主/女主穿着状态为……」。  
   - 脱外套之后各 phase 的提示词/情节说明中是否都有**男女主穿着状态**的明确一句。  
   - 人物形象参考图是否与当前 phase 的剧情阶段一致；若不一致，是否在提示词中强调「无外套/仅内搭」或使用阶段专用 ref。
