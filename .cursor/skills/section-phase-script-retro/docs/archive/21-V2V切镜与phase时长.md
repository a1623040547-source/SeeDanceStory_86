# 二十一、V2V 切镜边界与 phase 时长

## 1. Section 内 V2V 切镜边界（每最多 4 phase 一断）

**问题**：连续多段都用「上一段尾帧 → 本段首帧」的 V2V 衔接，会导致**画面噪声逐段扩散、视频质量越来越差**。

**规则**：在一个 section 内，**最多每 4 个 phase 就需要一次「切镜」**——保持环境/场景不变，但**不再使用 V2V**，下一段用新镜头/新起始画面（例如用环境参考图或本段首帧图重新起，而非以上一段尾帧为参考）。

**剧本设计**：在这些**交界处**（如 phase04 尾、phase08 尾等），剧本要有意安排：
- 段尾停留在**画面较为平稳的位置**（少大幅动作、少复杂光影变化），便于切镜后观感自然；
- 下一段（phase05、phase09 等）在 00/README 或提示词中标注为「切镜段：不用上段尾帧 V2V，用环境/首帧重新起」。

可与 [05-镜头过渡.md](05-镜头过渡.md) 中的「景别/镜头变化」「动作切点」等一起规划：在 phase 4、8、12… 的边界采用「非 V2V」的过渡方式。

---

## 2. Phase 时长不固定为 15 秒

**问题**：若默认每段都是 15s，容易为凑时长而稀释内容，或为压内容而挤满 15s，节奏与剧本不匹配。平台目前仅支持 **4～15s** 时长选择。

**规则**：**根据剧本自身的对话密度 / 内容密度来设定每个 phase 的时长**，不必每段都顶满 15s。平台目前**仅支持 4～15s**，在此范围内选择即可。例如：
- 对话密集、信息量大的段可以设计为较短（如 6～8s）；
- 留白、氛围、动作较慢的段可以设计为较长（如 12～15s）。

**剧本设计**：在 00 或 README 中为每个 phase 标注**建议时长**（在 4～15s 内，或「短/中/长」），并在情节说明与台词时间块中按该时长分配内容，避免「为 15s 而填满」或「为 15s 而压缩」。

与 [03-台词与节奏.md](03-台词与节奏.md) 中的快段/慢段、台词时间块配合使用。
