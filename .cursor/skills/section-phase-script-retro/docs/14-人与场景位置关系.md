# 十四、人与场景位置关系（配合初始化环境）

### 问题简述（中期复盘）

- **Seedance 对环境空间的位置关系理解有限**：在存在运镜（推/拉/摇/移等）时，容易让人物与场景的**相对位置**发生错误变化。
- 表现：例如人物本应在窗左侧、沙发前，运镜后却与窗户/沙发的关系错位；或多人相对位置在镜头运动中被「重新理解」而漂移。

### 改进要点

1. **在 phase 提示词中明确「人与场景的位置关系」**
   - 不只写「两人在客厅」，而要写清**人物在空间中的锚点**，例如：
     - 「男主站在窗左侧、面向室内；女主坐在沙发右侧，与男主相对。」
     - 「茶几在两人之间；镜头从男主肩后看向女主。」
   - 与**景别、运镜**一起写时，位置关系要一致：例如「中景，男主在画左、窗在画右背景；缓慢横摇向女主」。

2. **与「初始化环境」提示词配合**
   - 若本 section 有**第一段环境氛围参考图**或**环境初始化描述**（见 [09-环境氛围打光.md](09-环境氛围打光.md)），其中应包含**空间布局**的简要约定（如「沙发靠窗、茶几居中、门在画右」）。
   - 各 phase 提示词中的位置描述应与该**空间布局**一致，避免某段突然写成「沙发在画右」而前段是「沙发在画左」。

3. **运镜时显式约束「谁在画内、谁相对谁」**
   - 有推/拉/摇/移时，在提示词中补一句**运镜过程中保持不变的关系**，例如：
     - 「横摇时保持男主在画左、女主在画右，两人相对位置不变。」
     - 「拉远时茶几始终在两人之间，不改变人物与家具的相对位置。」

4. **通过尾帧观察空间关系**
   - 除在提示词中强调位置外，**写下一段提示词或上传参考前，应查看上一段尾帧**（如 `sectionXX_tail_NN.png`），确认人、物在画面中的**实际位置**（谁在画左/画右、与窗/门/家具的相对关系）。
   - 下一段提示词中的位置描述应与尾帧画面一致；若尾帧与剧本设想有偏差，以尾帧为准调整提示词，避免下一段生成时 Seedance 在错误的空间理解上继续漂移。

### 自检清单（位置关系）

- [ ] 每段提示词是否包含**人物在场景中的具体位置**（如窗左/沙发右/两人相对），而非仅「在客厅」「在室内」。
- [ ] 有运镜的 phase 是否在提示词中**显式约束**运镜过程中人与场景/人与人的相对位置不变（或按剧本写清合理变化）。
- [ ] 本 section 的**环境/空间初始化**（00 或第一段环境参考说明）是否约定了**空间布局**（沙发、窗、门、茶几等方位），各 phase 是否与之一致。
- [ ] 写下一段或上传参考前，是否**查看上一段尾帧**，根据尾帧中的实际人景位置撰写/调整本段提示词，保证空间关系延续。
