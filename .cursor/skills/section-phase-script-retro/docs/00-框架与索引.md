## 使用方式

- **新建 section**：按 01 → 02 → … → 06 的顺序过一遍各层自检清单；07 为与《剧本与视频生成流程》的衔接说明。
- **按问题查**：从 SKILL.md 的「何时使用」或本表「关注点」找到对应层级，再进入该文档。
- **原单题文档**：若需单独查阅某一小主题的原始复盘，见 [docs/archive/](archive/) 下按原编号保留的文件（如 03-台词与节奏.md、17-对话回应不跨phase.md 等）；**section03 对话重写**的完整复盘见 [archive/section03-对话重写复盘.md](archive/section03-对话重写复盘.md)，要点已并入 01 第 1.1 节与第 4 节。

## 合并说明（哪些可以合并）

- **01-剧本与台词**：台词节奏、动作/寡言人设、对话不跨 phase 同属「写什么、怎么说、谁反应」的内容层，合并后便于统一检查对白与节奏。
- **02-Phase 衔接与切镜**：镜头过渡、环环相扣、V2V 切镜与时长都是「段与段怎么接、何时断」的结构问题，放在一起便于做衔接与切镜规划。
- **03-场景环境与空间**：环境氛围、门与场景细致度、人景位置都是「场景长什么样、人在哪」的空间设定，合并后避免环境与位置矛盾。
- **04-运镜与画面约束**：景别运镜、画面人数、走路位移都是「镜头与画面怎么约束」的技术层，便于提示词一次写全。
- **05-跨 Phase 一致性**：穿着、物品/情节、禁止模糊表述都是「跨段不矛盾、表述唯一」的连贯层，合并后便于做跨段自检。
- **06-平台 15s 与素材**：全能参考与配乐都受平台 15s 限制，合并为同一「平台与素材」层。

原 07「与剧本流程衔接」保留为独立流程入口，不合并进其他主题。
