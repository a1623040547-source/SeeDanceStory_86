# 五、跨 Phase 一致性（穿着、物品、表述唯一）

本主题合并：**穿着/服装状态**、**物品关系与情节连贯**、**禁止模糊表述**。用于自检「跨段不矛盾、表述唯一」。

---

## 1. 穿着/服装状态跨 phase 一致性

### 问题简述

- 剧情中有**穿着变化**（如男主脱外套给女主）时，若只写「本段动作」而不写**当前穿着状态**，后续生成可能让角色恢复默认造型（如男主又穿上外套）。人物形象参考图中若角色穿着外衣，Seedance 也可能把 ref 里的外衣强加回角色身上。

### 改进要点

1. **在相关 phase 中显式写清穿着状态**：从「脱外套」发生的那一段之后，直至外套归还或场景结束，**每一段**的【本段情节说明】或提示词应包含一句**穿着锚定**（例：「本段内：男主已脱外套，只着内搭/衬衫；女主披着男主外套。」），可放在【本段动作与节奏】开场句或情节说明首句。
2. **人物形象参考图与剧情阶段匹配**：若 ref 中男主穿外套而剧情已进入「外套在女主身上」，可（A）为「脱外套后」单独准备无外套的男主 ref，或（B）不换 ref 时在提示词中**反复强调**本段造型（「男主未穿外套、仅内搭」「男主无外衣」）。

### 自检清单

- [ ] Section 内有明显穿着变化时，00 或 README 是否标出「从 phase N 起：男主/女主穿着状态为……」？
- [ ] 脱外套之后各 phase 的提示词/情节说明是否都有男女主穿着状态的明确一句？人物 ref 是否与当前 phase 剧情一致，若不一致是否在提示词中强调或使用阶段专用 ref？

## 2. 物品关系与情节连贯（跨 phase 状态一致）

### 问题简述

- **跨 phase 物品状态不一致**：某物品或动作在前一段已完成，后一段又重复描写（如 phase02 已「放好行李」，phase04 又写「把最后一件行李放好」），情节逻辑断裂。根因是写单段时未沿时间线检查前面各 phase 中该物品/动作是否已发生。

### 创作时的三类关系

- **位置关系**：人物在场景中的空间位置、与景/物的相对关系（见 [03-场景环境与空间.md](03-场景环境与空间.md)）。
- **情节关系**：前后段情节先后、情绪与节奏衔接（见 [02-Phase衔接与切镜.md](02-Phase衔接与切镜.md) 环环相扣）。
- **物品关系**：**物品在剧情中的状态与归属**（行李是否已放下、帽子是否已拿等）、**动作是否已在更早的 phase 完成**，避免同一动作或物品状态在多段中重复或矛盾。

### 改进要点

1. **建立「物品/动作时间线」**：在 section 的 00 或 README 对关键物品/动作做简要标注（如「行李：phase02 门边放好；帽子/包：phase04 末拿、phase05 出门」）；写每一段时先扫一眼：本段要写的动作或物品是否已在更早的 phase 完成，若已完成则本段不再写「再次完成」或改为其它收束动作。
2. **情节说明与提示词显式承接「物品状态」**：若前段已有「放好行李」「放下杯子」「拿起包」等，本段应写「承接上段行李已放好/包已拿起…」，本段动作只写**之后**发生的事。
3. **写新 phase 前的自检**：写 phase N 时快速过一遍 phase 1～N-1 的【本段情节说明】与【本段动作与节奏】，列出「已出现的物品与已完成的动作」清单。

### 自检清单

- [ ] 本段是否写了某件「放/拿/用」类动作，而该动作或该物品状态已在**更早的 phase** 完成？若有，是否已删除或改写重复动作？
- [ ] Section 内是否有关键物品/动作的简单时间线或标注？本段情节说明是否与「承接上段…」一致（若上段已写「放好行李」，本段不应再出现「把行李放好」）？

## 3. 禁止模糊表述（明确指向）

**规则**：在情节说明、位置关系、物品关系、提示词等**任何涉及剧情设计的位置**，都不要使用「A 或 B」「A/B」等模糊描述，必须**明确指向一种**（是 A 就是 A），否则不同镜头或不同 phase 中会做出不一致选择，导致画面错乱或前后矛盾。

### 反例与正确做法

- 反例：「蕾娜拿**帽子或小包**」「手持**帽或包**」→ 可能这段拿帽、下段拿包或同段前后不一致。改进：全 section 统一为**帽子**（或小包），情节说明、物品、提示词一律写「帽子」「持帽子」，不写「帽或包」。
- 物品：确定是帽子就写「帽子」，是小包就写「小包」；不在同一句里写「帽子或包」。动作/位置：确定一种写法，不写「向放帽/包处走」或「持帽或包」。若创作阶段未定稿，先在 00 或 README 定稿（如「本 section 蕾娜手持物为帽子」），再在各 phase 中统一使用。

### 自检清单

- [ ] 本段是否有「A 或 B」「A/B」类表述？若有，是否已改为唯一确定项并与其它 phase、与 00 一致？
- [ ] 跨 phase 的同一物品/动作是否全 section 用词一致（如都是「帽子」而非有的写「帽」有的写「帽或包」）？
