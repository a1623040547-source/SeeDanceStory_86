# 海边民宿室内（玄关/带窗房间）

- **名称/代号**：民宿室内、海边民宿一隅（story 海边度假 S1）
- **场景**：海边小镇民宿室内；玄关或带窗房间一体；窗框占画面一侧或背景，窗外可见海平线与天空；室内木质地板或白墙、简洁家具（门侧矮柜、衣架等）。
