# 浅滩 / 水边

- **名称/代号**：浅滩、水边（story 海边度假 S2）
- **场景**：沙滩与海水交界，浪花没过脚踝高度；清澈见底或浅蓝绿色；日照、水面反光与泡沫。
- **光线**：日照，高亮与阴影明确；水面反光。
- **氛围**：明亮、活泼、水花飞溅感；86 动画风、夏日、治愈。
- **空间锚点**：海水在脚下、没过脚踝；背后或一侧为沙滩；画面内仅两人、无其他人物。
- **环境提示词（【场景与氛围】）**：浅滩/水边，日照与水面反光；浪花没过脚踝，清澈见底或浅蓝绿色；沙滩与海水交界；明亮、活泼、水花飞溅感。
- **用于生成环境图**：浅滩/水边：清澈海水没过脚踝高度，水面反光与泡沫；沙滩与浅水交界；日照、明亮；可带浪花飞溅或涟漪；86 动画风、活泼、夏日感。
