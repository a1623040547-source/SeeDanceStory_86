# 日间沙滩

- **名称/代号**：日间沙滩、沙滩日间（story 海边度假 S3）
- **场景**：日间沙滩全景或中景；蓝天、白云、远处海平线；前景或中景为细沙与少量浪花；脚边浪花与泡沫。
- **光线**：阳光充足、高对比；光线高亮、阴影清晰。
- **氛围**：开阔、治愈、略带海风感；86 动画风、略有过曝感。
- **空间锚点**：背后或一侧为海；两人在沙滩上、沿沙滩走；画面内仅两人。
- **环境提示词（【场景与氛围】）**：日间沙滩，蓝天白云，阳光充足；海浪、细沙、远处海平线；脚边浪花与泡沫；光线高亮、阴影清晰；开阔、治愈、略带海风感。
