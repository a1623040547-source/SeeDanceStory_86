# 主角身份文档（86 辛恩 × 蕾娜）

角色基于**《86-不存在的战区》**原作，本文件为两人**角色精髓**的合并版，供剧本、提示词与角色一致性使用。**不绑定具体故事或世界观**；时间线、服装、关系阶段等由各 story/section 的 00 或总述单独设定。详细与配图见 @character-shin、@character-lena。

---

## 一、辛恩（Shin / シン）

| 项目 | 内容 |
|------|------|
| **身份** | 86 区出身，少年兵经历；幸存者，背负亡者之声与决意前行 |
| **性格** | 沉稳、少言、内敛；多用行动与选择表达，寡言≠无反应、≠无情感 |
| **表达方式** | 台词极简；情绪通过**微反应**传达（视线、眼珠微动、喉结/眉/嘴角极微动、手指/肩/呼吸） |
| **声线** | 低沉、平静，偶有淡淡笑意 |
| **外貌** | 遵循 86 原作动画设定：青年、短发、锐利与疲惫并存的眉眼；便服多为深色或军装风 |
| **与蕾娜** | 对她有深层信任与未说破的在意；**在她身边时气质会柔和、带笑意**；会默默照顾、不躲开亲近；被逗时会有极轻微反应（眉动、喉结动、微微笑）；不主动多话但会接话或简短问句 |
| **禁忌/注意** | 避免写成「无表情、无互动」；与蕾娜同框时须体现**温柔、带笑意**；提示词中须显式写「对蕾娜的视线/微表情/肢体回应」与触发–反应链 |

## 二、蕾娜（Lena / レーナ）

| 项目 | 内容 |
|------|------|
| **身份** | 前共和国军官；理想主义、为 86 发声；与辛恩等人相遇后的幸存者与同行者 |
| **性格** | **独立、眼神坚毅、有指挥官气质**；开朗、坚韧、会撒娇也会较真；先开口多、表情丰富，情感外放 |
| **两面性** | 平日独立可靠、有决断；**仅在辛恩面前**会流露**女孩子的一面**（照れ、撒娇、柔软、小声答应等）。勿写成全程软萌——柔软与照れ是在辛面前的特别表现。 |
| **表达方式** | 台词有情绪起伏（赞叹→侧头看人→愣住→满足→调侃→安静）；可带气声、轻笑、欲言又止 |
| **声线** | 柔和、随情绪有轻重与语气变化 |
| **外貌** | 遵循 86 原作动画设定：银发（或设定中的发色）、少女感；与辛恩同框时比例与画风一致 |
| **与辛恩** | 会追问、调侃、偶尔欲言又止；收到照顾时会愣一下再道谢或小声嘀咕；安静时也会靠过去、看侧脸；信任他、会率先表达关心；**在他面前才会显露出照れ、撒娇、「既然是辛君说的那就…」等女孩子的一面** |
| **禁忌/注意** | 避免台词平均化；可有「快段」与「慢段」、留白与密集对话交替 |
