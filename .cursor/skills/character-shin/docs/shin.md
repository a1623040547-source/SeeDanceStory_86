## 身份与性格（原作精髓）

| 项目 | 内容 |
|------|------|
| **身份** | 86 区出身，少年兵经历；幸存者，背负亡者之声与决意前行 |
| **性格** | 沉稳、少言、内敛；多用行动与选择表达，寡言≠无反应、≠无情感 |
| **表达方式** | 台词极简；情绪通过**微反应**传达（视线、眼珠微动、喉结/眉/嘴角极微动、手指/肩/呼吸） |
| **声线** | **低沉、深沉、带磁性**；平静，偶有淡淡笑意。避免过于年轻化、少年音——要有重量感与阅历感，而非清脆或稚嫩 |
| **外貌** | 遵循 86 原作动画设定：青年、短发、锐利与疲惫并存的眉眼；便服多为深色或军装风 |
| **与蕾娜** | 对她有深层信任与未说破的在意；**在她身边时气质会柔和、带笑意**（不必大笑，嘴角/眼神柔和即可）；会默默照顾、不躲开亲近；被逗时会有极轻微反应（眉动、喉结动、微微笑）；不主动多话但会接话或简短问句 |
| **禁忌/注意** | 避免写成「无表情、无互动」；与蕾娜同框时须体现**温柔、带笑意**（嘴角/眼神柔和），而非冷淡；提示词中须显式写「对蕾娜的视线/微表情/肢体回应」与触发–反应链。**视频/配音**：声线勿过于年轻化，须保留低沉、深沉与磁性。 |

- **画风**：86 原作动画风；与项目 refs/characters 中角色图一致即可。

- **服装**：由具体作品/故事设定决定；同一叙事单元内服装状态须一致。
